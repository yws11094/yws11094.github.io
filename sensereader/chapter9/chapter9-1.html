Ⅸ-1-1. Braillenote 지원 기능


센스리더는 Braillenote 한국어판과 완벽한 연결 기능을 제공합니다.

센스리더와의 점자 입출력 기능은 물론 점형의 사용자 지정, 독서기를 통한 파일 전송 등의 다양한 연결 기능을 지원함으로써, 정보 습득이나 학습, 업무 등의 영역에서 효율적인 작업이 가능할 수 있도록 도움을 줍니다.

센스리더의 Braillenote 지원 기능은 다음과 같습니다.

1. 센스리더의 강력한 화면읽기 기능을 음성출력과 더불어 점자정보단말기를 통해 점자로 확인할 수 있습니다.
2. 한자나 특수문자의 점자 출력시 사용자가 출력 점형을 지정할 수 있습니다.
3. 점자정보단말기에서 센스리더의 다양한 기능과 윈도우즈의 각종 명령들을 실행할 수 있습니다.
4. 시리얼 케이블과 USB 케이블 및 블루투스 등의 다양한 연결방법을 제공합니다.
5. Braillenote 한국어판 자체의 인터넷 기능과 더불어 ActiveSync를 통한 인터넷 연결기능을 제공합니다.
6. 센스 독서기를 이용한 손쉬운 파일전송기능을 지원합니다.


Ⅸ-1-2. Braillenote 연결 방법


센스리더는 시리얼 케이블과 블루투스 통신을 통해 점자정보단말기로 점자를 표시하며, 점자정보단말기에서 센스리더의 기능을 제어하거나 윈도우즈의 명령을 실행할 수 있는 기능을 제공합니다.
시리얼 케이블을 이용한 통신을 하기 위해서는, Braillenote 한국어판 구입시 제공되는 시리얼 케이블을 컴퓨터와 Braillenote의 시리얼 포트 단자에 연결해 주어야합니다. 시리얼 통신은 케이블을 연결하는 것만으로 모든 준비과정이 끝나기 때문에 설치가 쉽다는 장점은 있으나 Braillenote와의 통신을 위해서는 매번 케이블을 연결해 주어야 하는 불편함이 있기 때문에 Braillenote를 자주 이동해서 사용하는 사용자라면 블루투스를 통한 통신이 더 효율적일 수 있습니다. 　

다음의 과정에 따라 Braillenote와 센스리더를 연결합니다.
1. 시리얼 케이블을 컴퓨터와 Braillenote의 시리얼 단자에 연결합니다. 이 때, 사용자가 컴퓨터의 어느 컴포트에 케이블을 연결했는지 정확하게 확인합니다. 펜티엄 4 이상의 최신 기종들은 대부분 Com1만을 지원하는 경우가 많습니다.
블루투스 통신을 사용하기 원하는 사용자는 컴퓨터에 USB Dongle을 설치한 다음 Braillenote와 블루투스 설정을 해 주어야 합니다.
Braillenote의 블루투스 설정에 관해서는 Braillenote 메뉴얼을 참고하시고, USB Dongle 설정은 제조사별 메뉴얼을 참고하십시요.
2. <Ctrl-\> 키를 눌러 센스리더 메인창을 연 다음, [일반] -> [점자디스플레이 설정] 메뉴를 실행합니다.
3. '점자디스플레이 설정' 대화상자가 열리면 다음 항목들의 설정을 합니다.
1) 스페이스바를 눌러 '장치 사용' 체크상자를 선택합니다.
2) 장치 목록 중 현재 컴퓨터에 연결되어 있는 점자 디스플레이 장치를 선택합니다.
3) 컴포트를 설정하는 목록상자로 이동하여 현재 사용 중인 컴포트에 맞춥니다.

4) '확인' 버튼을 눌러 설정을 적용합니다.
4. BrailleNote의 주메뉴에서 [스크린리더 연결] 메뉴를 선택합니다.
5. [연결 포트' 프롬프트에서 컴퓨터와 연결된 통신 종류를 선택합니다. 현재는 '시리얼'과 '블루투스'의 두 가지 방식을 지원합니다. 설정값을 변경하기 위해서는<Space-3-4> 코드를 누릅니다.
* 만약 점자정보단말기에 정보가 올바르게 표시되지 않는다면 센스리더를 종료한 후 다시 실행합니다.


Ⅸ-1-3. ActiveSync 설치


브레일노트와 데스크탑 사이에 데이터 파일을 전송하거나 브레일노트 인터넷 기능을 활용하기 위해서는 ActiveSync 프로그램을 데스크탑에 설치한 후, 브레일노트 연결 설정을 해 주어야 합니다.

다음 과정에 따라 ActiveSync 프로그램을 설치합니다.
① 브레일노트 보급시 같이 제공된 'activesync38.exe' 파일을 컴퓨터에 복사한 후, <Enter> 키를 눌러 실행합니다.
ActiveSync 프로그램은 통신망이나 인터넷 자료실 등에서도 쉽게 구할 수 있습니다.
② 설치 파일 실행 후 잠시 기다리면 설치 안내 대화상자가 출력됩니다. '다음' 버튼을 눌러 다음 단계로 진행합니다.
③ '설치 경로 확인' 대화상자가 나타나면 '다음' 버튼을 눌러 역시 다음 단계로 진행합니다.
설치 파일을 설치하는 동안 약간의 시간이 경과됩니다.
④ '연결 시작' 대화상자가 출력되는 것을 확인합니다.
시스템에 따라서는 재부팅 메세지를 출력하는 경우도 있습니다. 이 때는 '예' 버튼을 누릅니다. 재부팅 이후 다시 '연결 시작' 대화상자가 다시 나타납니다.
⑤ 브레일노트와 컴퓨터를 케이블로 연결한 후, 브레일노트의 전원을 켭니다.
⑥ 잠시 후 경쾌한 배경음이 들리면서 '새 파트너 관계 설정' 대화상자가 나타나면 정상적으로 장치가 연결된 것입니다. '새 파트너 설정' 대화상자는 '아니오' 라디오 버튼을 선택한 뒤, '확인' 버튼을 눌러 창을 닫아줍니다.
⑦ 바탕화면의 '내 컴퓨터'를 실행한 다음, '모바일 장치'에서 'Enter' 키를 누르면 브레일노트의 폴더들이 표시됩니다.
이후의 파일 복사 및 삭제, 이동 등의 파일 관리 방법은 일반적인 데스크탑에서의 파일 작업과 동일합니다.

* 브레일노트의 전원을 켰을 때 '새 파트너 관계 설정' 대화상자가 나타나지 않는다면 브레일노트의 전원을 한 번 끈 후 다시 켭니다. 만일, 전원을 새로 켠 이후에도 장치가 연결되지 않는다면 브레일노트 전원 스위치 위쪽에 있는 리셋 버튼을 한 번 눌러줍니다.


Ⅸ-1-4. ActiveSync를 통한 인터넷 연결


BrailleNote 한국어판은 인터넷에 연결할 수 있는 다양한 방법을 제공합니다.

사용자는 각 환경에 알맞는 인터넷 연결 방법을 선택함으로써 인터넷을 좀 더 폭넓게 즐길 수 있습니다.

BrailleNote 한국어판에서 지원하는 인터넷 연결 방식은 다음과 같습니다.
1. 유/무선 랜을 이용한 단말기 자체의 인터넷 기능(랜카드 별도 구매)
2. 센스리더와 연결을 통해 윈도우즈 인터넷 익스플로러를 직접 실행하여 활용하는 방법
3. ActiveSync를 통한 인터넷 연결 기능

다음의 과정에 따라 ActiveSync를 통한 인터넷 연결 설정을 합니다.
① BrailleNote의 주메뉴에서 [옵션] -> [연결 설정] -> [새 연결] 항목을 차례대로 실행합니다.
② 새로운 연결 이름을 사용자 임의로 입력하고 <Enter> 키를 누릅니다.
③ 장치선택 프롬프트가 나타나면 <Space-3-4> 키를 눌러 '유선랜'을 선택하고 <Enter> 키를 누릅니다.
④ 아이피 프롬프트는 '자동으로 받기'로 설정합니다.
⑤ <Space-E> 코드를 누르면 "레코드를 추가하시겠습니까?"라고 묻게 되는데 'Y' 키를 누릅니다.
⑥ ActiveSync 프로그램을 이용하여 컴퓨터와 브레일노트를 연결합니다.
⑦ BrailleNote의 주메뉴에서 인터넷을 실행합니다. 연결 방식을 방금 만들었던 연결 이름으로 지정하고 <Enter> 키를 눌러 인터넷에 연결합니다.

* 이 기능을 사용하기 위해서는 컴퓨터가 인터넷에 연결된 상태여야 합니다.

Ⅸ-1-6. 센스 독서기 파일 전송


센스 독서기는 데스크탑과 브레일노트 사이의 데이터 전송 작업을 보다 편리하게 수행할 수 있도록 손쉬운 각 장치별 이동 기능과 브레일노트 파일 형식의 변환 복사 기능을 지원합니다.
단, 이 기능을 정상적으로 사용하기 위해서는 ActiveSync 프로그램을 이용하여 데스크탑과 브레일노트가 먼저 연결되어 있어야 합니다.

1. 데스크탑과 브레일노트의 이동
파일 전송 작업을 위해 데스크탑과 브레일노트 사이를 이동하기 위해서는 탐색창에서 '>' 키를 누르거나, 원하는 드라이브 목록에서 <Enter> 키를 누릅니다. ActiveSync가 설치된 경우, 브레일노트는 드라이브 목록 하단에 '모바일 장치'로 표시됩니다.
센스 독서기는 각 장치가 변경될 때마다 현재의 장치 종류를 음성으로 안내해 줍니다.

2. 데이터 파일의 전송
브레일노트에 있는 파일을 데스크탑으로 복사하거나, 데스크탑의 존재하는 파일을 브레일노트로 복사하는 작업은 일반적인 센스 독서기의 파일 관리 사용법과 동일합니다.
또한, 센스 독서기는 기본적인 파일 관리 기능과 함께 파일 복사시 다양한 문서 파일(txt, hwp, doc, html, xls, ppt)을 사용자가 지정한 파일 형식으로
변환하여 브레일노트로 전송하고, 브레일노트의 .kwt 파일을 일반 텍스트로 변환하여 데스크탑에 복사할 수 있는 변환 기능을 제공합니다.
센스 독서기의 기본값은 원본 파일 형식을 그대로 복사는 것이며, 이 기능을 사용하기 위해서는 센스 독서기에서 파일 변환 설정을 먼저 해 주어야 합니다.
