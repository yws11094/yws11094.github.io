ⅩⅦ-1. 환경 설정 이해


프로페셔널과 더 넥스트는 MS-OFFICE, 한글 워드프로세서에 대해 각 프로그램별 특성에 맞는 환경을 구성하고 있습니다.
기본적으로 설정된 환경을 사용자가 변경하고자 한다면, <Ctrl-Shift-F9> 키를 눌러 설정 대화상자를 호출하고 원하는 설정으로 변경해 주면 됩니다.

오피스 및 한글 워드프로세서의 환경 설정은 센스리더의 구두점 설정 방법과 유사합니다.
즉, 여러 개의 설정 패턴을 만들어 놓고 그 중 한 가지를 선택하여 사용하는 방식입니다.
설정' 대화상자가 열리면 처음 포커스는 목록 상자에 위치하게 됩니다.
목록은 '자동 반응 설정, 키 반응 설정, 디폴트 자동 반응 설정, 디폴트 키 반응 설정 등 4개의 항목으로 구성되어 있습니다.
자동 반응 설정이란 캐럿의 움직임, 페이지 변경, 셀 위치 변경, 시트 변경 등과 같은 이벤트에 대한 기능 설정을 의미하고, 키 반응 설정은 지정된 기능키에 대한 기능의 설정을 나타냅니다.
디폴트 자동 반응 설정과 디폴트 키 반응 설정은 열려진 문서의 환경이 없을 때 적용되는 환경을 의미합니다.

엑셀의 설정 대화상자를 열고 자동 반응 설정에 위치한 다음 <Space> 키를 누르면 기본, 기본(제목셀읽기), 내용만 읽기, 행읽기 등 4개의 설정이 차례대로 선택되는 것을 확인할 수 있습니다.
이 4개의 설정 중 사용자가 작업에 맞는 설정을 선택하고 <확인> 버튼을 누르면 현재 선택된 설정이 그대로 적용됩니다.
예를 들어, 엑셀 문서에서 데이터를 입력하거나 일반적인 작업에서는 기본 설정을 사용하고, 항목이 많은 문서에서 데이터를 입력할 경우 기본(제목셀읽기) 설정을 사용하면 작업을 좀 더 효율적으로 진행할 수 있습니다.
또한 내용만 읽기, 행읽기 설정은 기존 문서의 내용을 쉽게 파악할 수 있도록 도움을 줍니다.

'설정' 대화상자의 설정 목록에서 <Tab> 키를 누르면 기능 목록과 세부 옵션 목록이 차례대로 나옵니다.
세부 옵션의 조정은 '설정 목록' 대화상자를 통해서도 가능하지만, '설정' 대화상자에서 세부 옵션을 변경할 경우 해당 설정에 지정된 연결 기능 전체에 변경된 값을 지정할 수 있습니다.
'설정' 대화상자의 기능 목록은 전체 기능 중에서 세부 옵션 조정이 자주 사용되는 기능만을 표시합니다.

주의1: 기본 설정은 센스리더의 기본 환경처럼 환경 전체를 포괄하는 개념이 아니고 단순히 설정을 구별하기 위한 이름입니다.
주의2: 기능이란 센스리더가 개별 프로그램의 정보를 확인하여 음성출력할 수 있는 하나 하나의 동작을 의미합니다. 엑셀의 경우 셀 포인터 내용 읽기, 주소 읽기, 서식 읽기, 이름 읽기, 메모 읽기 등 약 70여 가지의 기능을 가지고 있습니다.
연결 기능은 전체 기능 중에서 이벤트 발생이나 기능키 실행에 대해 실제 기능이 동작할 수 있도록 선택되어 있는 기능을 의미합니다.  

사용자는 미리 제공된 설정들을 편의에 맞도록 변경하거나 삭제, 추가 등의 작업을 할 수 있습니다. 자동 반응 설정에 위치한 후, <POPUP> 키를 누르면 설정 목록, 설정 이름 변경, 설정 삭제 등의 메뉴가 나타납니다. 설정 이름을 변경하고자 하거나 삭제를 원한다면 해당 메뉴를 실행하면 됩니다.

설정 목록 메뉴는 각 설정에 연결된 기능을 구성할 수 있는 메뉴입니다.
설정 목록 메뉴를 실행하면 '설정 목록' 대화상자가 출력되는데, 대화상자에는 이름 목록상자, 이벤트 목록상자, 연결 기능 목록상자, 세부설정 목록상자 등 4개의 컨트롤이 배치되어 있습니다.
만일, 사용자가 기본 설정에서 셀 포인터 이동시 특정 기능을 추가하거나 삭제하고 싶다면 이름 목록상자와 이벤트 목록상자에서 각각 해당 항목을 선택하고 연결 기능 목록상자에서 원하는 기능을 추가/삭제하면 됩니다.
모든 설정을 마친 후에는 반드시 <확인> 버튼을 눌러야 변경된 설정값이 적용됩니다.


ⅩⅦ-2. 환경 설정 기본


아래는 환경 설정 방법을 세부적으로 정리한 것입니다. 각 프로그램의 이벤트, 기능 목록은 해당 프로그램별 챕터를 참고하여 주시고, 실제 환경 설정 예제들이 같이 포함되어 있으니 환경 설정 방법을 이해하시는데 도움이 될 것입니다.


A. 동작 원리

1. 센스리더 기본환경설정과는 별개로 오피스 지원을 위한 별도의 환경파일을 가지며, 센스리더의 오피스 지원 동작은 환경설정에 의해 제어됩니다.
2. 환경설정은 크게 자동 반응(이벤트)과 키 반응(단축키)로 구분되며 두 설정은 서로간에 영향을 주지 않습니다.
3. 여러 개의 서로 다른 설정을 만들어 놓고 문서마다 특정 설정과 연결하여 사용할 수 있습니다.
4. 문서와 연결된 자동 반응 설정이 없는 경우 디폴트 자동 반응 설정이, 연결된 키 반응 설정이 없는 경우 디폴트 키 반응 설정이 적용됩니다.


B. 설정 대화상자 호출

1. 설정을 원하는 오피스 메인 윈도우를 활성화합니다.
2. <Ctrl-Shift-F9> 키를 누릅니다.
3. 각 프로그램(엑셀, 워드, 파워포인트) 설정 대화상자가 호출됩니다.


C. 저장된 설정 선택

1. 설정 대화상자를 호출합니다.
2. 목록에서 변경을 원하는 항목을 선택합니다.
3. 원하는 설정 이름이 선택될 때까지 'Space'키를 누릅니다.


D. 설정 대화상자에서의 세부 옵션 조정

1. 설정 대화상자를 호출합니다.
2. 자동 반응 혹은 키 반응 설정 중 원하는 항목을 선택합니다.
3. <Tab> 키를 눌러 기능 목록상자로 이동한 다음 원하는 기능을 선택합니다.
4. <Tab> 키를 다시 눌러 세부 옵션 목록까지 이동합니다.
5. 변경을 원하는 세부 옵션 항목을 선택하고 <Space> 키를 눌러 설정/해제를 변경합니다.
6. <확인> 버튼을 눌러 변경 사항을 적용합니다.


E. 설정 추가, 삭제, 이름변경

1. 설정 대화상자를 호출합니다.
2. 자동 반응 혹은 키 반응 설정 중 원하는 항목을 선택합니다.
3. 팝업키를 눌러 메뉴를 호출합니다.
4. [설정 목록...]메뉴를 실행합니다.
5. 설정 목록 대화상자가 호출되면 '이름'목록상자로 이동합니다.(기본적으로 포커스 되어 있음.)
6. 필요할 경우(추가, 삭제, 이름 변경) 대상항목을 선택한 후 팝업메뉴를 호출합니다.
7. 원하는 메뉴 항목을 실행한 후 필요한 추가 동작을 해 줍니다.
(추가의 경우 '복사할 설정' 목록에서 원하는 설정을 골라 추가되는 설정에 복사하여 필요한 부분만을 변경하여 추가할 수 있습니다.)


F. 기능키 추가, 삭제, 변경

1. 설정 대화상자를 호출합니다.
2. 키 반응 설정을 선택한후 팝업 메뉴를 호출합니다.
3. 설정 목록메뉴를 실행하여 키 반응 설정 목록 대화상자를 호출합니다.
4. '이름'목록상자에서 설정 이름을 선택한 후 '기능키'목록상자로 이동합니다.
5. 필요할 경우(삭제, 변경) 대상 항목을 선택한 후 팝업 메뉴를 호출합니다.
6. 원하는 메뉴 항목을 실행 한 후 필요한 추가 동작을 해 줍니다.


G. 기능 연결

1. 설정 대화상자를 호출합니다.
2. 자동 반응 설정과 키 반응 설정 중 변경을 원하는 항목을 선택한후 팝업 메뉴를 호출합니다.
3. 설정 목록메뉴를 실행하여 반응 설정 목록 대화상자를 호출합니다.
4. '이름' 목록상자에서 설정 이름을 선택합니다.
5. 자동 반응의 경우 '이벤트' 목록상자로, 키 반응의 경우 '기능키' 목록상자로 이동한 후 기능 연결 상태를 변경할 항목을 선택합니다.
6. '연결기능' 목록상자로 이동한 후 팝업 메뉴를 호출합니다.
7. '기능 구성' 대화상자가 나타나면 기능 연결 상태를 원하는 형태로 변경합니다.
1) '전체기능' 목록에서 기능을 선택한 후 'Space'를 누르면 선택한 기능이 '연결기능' 목록에 추가됩니다.
2) '연결기능' 목록에서 기능을 선택한 후 'Space'를 누르면 선택한 기능이 '연결기능'에서 삭제됩니다.
3) '기능 구성'이 키 반응 설정에서 호출된 경우 기능분리기호('----------')가 '전체기능'에 추가됩니다.
4) 기능분리기호로 분리된 기능은 기능키를 연속하여 누른 경우 차례로 실행됩니다.


H. 기능 순서 변경

1. 설정 대화상자를 호출합니다.
2. 자동 반응 설정과 키 반응 설정 중 변경을 원하는 항목을 선택한후 팝업 메뉴를 호출합니다.
3. 설정 목록메뉴를 실행하여 '반응 설정 목록' 대화상자를 호출합니다.
4. '이름' 목록상자에서 설정 이름을 선택합니다.
5. 자동 반응의 경우 '이벤트' 목록상자로, 키 반응의 경우 '기능키' 목록상자로 이동한 후 기능 연결 상태를 변경할 항목을 선택합니다.
6. '연결기능' 목록상자로 이동한 후 이동을 원하는 기능을 선택합니다.
7. <Alt-위화살표>(위로이동), <Alt-아래화살표>(아래로 이동)를 사용하여 기능의 순서를 변경합니다.
** '기능 구성' 대화상자의 '연결기능' 목록에서도 같은 키를 사용하여 기능의 순서를 변경할 수 있습니다.


I. 세부 옵션 변경

1. 설정 대화상자를 호출합니다.
2. 자동 반응 설정과 키 반응 설정 중 변경을 원하는 항목을 선택한후 팝업 메뉴를 호출합니다.
3. 설정 목록메뉴를 실행하여 '반응 설정 목록' 대화상자를 호출합니다.
4. '이름' 목록상자에서 설정 이름을 선택합니다.
5. 자동 반응의 경우 '이벤트' 목록상자로, 키 반응의 경우 '기능키' 목록상자로 이동한 후 기능 연결 상태를 변경할 항목을 선택합니다.
6. '연결기능' 목록상자로 이동한 후 옵션을 변경할 기능을 선택하고 '세부설정' 목록상자로 이동합니다.
7. 변경할 옵션을 선택하고 <Space> 키를 눌러 설정값을 변경합니다.
