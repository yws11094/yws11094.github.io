ⅩⅣ-6-1. 목록 구조


탐색창은 현재 폴더 내의 항목들을 목록으로 표시해 주며, 사용자는 이들 목록에서 원하는 항목을 찾아 필요한 작업을 실행합니다. 목록을 표시할 때는 폴더, 파일, 드라이브 순으로 표시하게 됩니다. 이 순서는 정렬 기준에 따라 앞선 항목이 있다 하더라도 뒤섞여서 출력되지는 않습니다.

1. 폴더 목록

폴더는 폴더 이름 뒤에 '\' 문자를 붙여 폴더임을 표시합니다.

폴더 항목 중 가장 처음에 나오는 '..' 항목은 현재 폴더의 상위 폴더를 의미하며, 루트 폴더를 제외한 모든 폴더에 존재합니다. 상위
폴더('..\')는 현재 폴더의 상위 폴더로 이동할 때 사용합니다.

2. 파일 목록

폴더 목록 이후에 파일 목록이 출력됩니다. 파일 목록에는 파일 이름과 확장자가 표시되며, 현재 폴더에 파일이 존재하지 않는 경우 폴더 목록 다음에 바로 드라이브 목록이 나타납니다.

3. 드라이브 목록

드라이브 목록은 드라이브 이름 다음에 ':' 문자를 붙여 드라이브임을 표시합니다.

현재 컴퓨터에서 사용 가능한 드라이브 항목을 표시하며, 항상 알파벳 순서로 표시하게 됩니다.


ⅩⅣ-6-2. 항목 선택 및 사용


센스 독서기의 탐색창에서 어떤 작업을 하기 위해서는 먼저 작업의 대상이 되는 항목에 위치를 옮겨 주어야 하며, 이후 다양한 방법으로 작업을 실행할 수 있습니다. 다음은 항목의 이동 방법과 사용에 관한 설명입니다.

1. 이동키를 이용한 항목의 이동

① 아래, 위 화살표: 선택을 아래, 위로 이동합니다.

② Home키: 현재 그룹(폴더, 파일, 드라이브) 앞에 다른 그룹이 있으면 그 그룹의 제일 마지막 항목으로 선택을
옮기고, 그렇치 않으면 현재 그룹의 첫 항목을 선택합니다.

③ End키: 현재 그룹 뒤에 다른 그룹이 있으면 그 그룹의 첫 항목을, 그렇지 않은 경우에는 현재 그룹의 마지막 항목을 선택합니다.

④ PGUP, PGDN: 한 화면씩 위, 아래로 선택을 이동합니다.

2. 문자 입력에 의한 항목 이동

독서기에서는 찾고자 하는 항목 이름을 직접 타이핑하여 항목을 찾는 기능을 지원합니다.

① 문자가 입력되면 그 문자로 시작되는 항목을 찾습니다.
항목이 찾아지면 그 항목으로 선택을 옮깁니다.

② 문자가 다시 입력되면 앞에 입력했던 문자 뒤에 현재 입력된 문자를 덧붙여 찾습니다. 항목이 찾아지면 그 항목으로 선택을 옮기고 만일 찾아지지 않았다면 현재 입력된 문자를 처음 문자로 하여 ①번 과정을 수행합니다.

③ 문자가 계속 입력되면 ②번 과정을 반복합니다.

3. 항목의 이름 확인

어떤 항목에서 왼쪽, 오른쪽 화살표키를 누르면 해당 항목의 이름을 한 글자씩 확인할 수 있습니다.

4. 기본 동작의 실행

기본 동작이란 어떤 항목에서 <Enter>키를 누르는 것을 의미합니다. 항목의 종류에 따라 다음과 같이 실행됩니다.

① 항목이 파일인 경우 확장자 연결이 된 프로그램을 찾아 실행합니다. 확장자 연결이 되어 있지 않다면 윈도우즈 탐색기와 동일한 동작을 합니다.

② 항목이 폴더라면 해당 폴더로 이동합니다.

③ 항목의 종류가 드라이브인 경우 해당 드라이브로 이동합니다.

드라이브로의 이동은 이동하려는 드라이브 명을 대문자로 입력하는 것으로도 가능합니다. 그렇기 때문에 반드시 드라이브를 선택할 필요는 없습니다. 만일 'c' 드라이브로 이동하고자 할 때에는 언제나 대문자로 'C'만 입력하면 바로 이동됩니다.

5. 옵션을 지정한 실행

프로그램(*.exe)에서 <Enter>키를 누르면 해당 프로그램이 실행됩니다. 그러나 프로그램 실행시 옵션을 주어야 하는 경우에는
<Shift-Enter>키를 누릅니다. '옵션 입력' 대화상자가 나타나면 프로그램 실행시 필요한 옵션을 입력합니다.

6. 윈도우즈 탐색기의 연결 프로그램 사용

독서기의 확장자 연결 기능을 사용하지 않고 윈도우즈 탐색기의 연결 프로그램을 직접 사용하려면 <Ctrl-Enter>키를 누릅니다.

예를 들어, 기본적으로 텍스트 파일의 경우 센스 독서기에서는 독서창(xvbrd.exe)과 연결되어 있지만 윈도우즈 탐색기에서는 메모장
프로그램과 연결되어 있습니다. 독서기에서 메모장을 실행하여 텍스트 파일을 열고자 할 때 이 기능을 사용합니다.


ⅩⅣ-6-3. 다중 선택 및 선택 해제


독서를 하거나 복사 및 이동 등의 작업 대상이 되는 파일이 여러 개일 경우 해당 파일들을 모두 선택해 주어야 합니다.

독서기에서는 다음의 방법을 사용하여 다중 선택을 할 수 있습니다.

1. <Shift-화살표>키를 이용한 다중 선택

<Shift>키를 누른 채 방향을 이동하면 이동된만큼의 모든 항목들이 선택됩니다. 윈도우즈 탐색기와는 달리 선택된 역방향으로 이동했을 때에도 선택은 해제되지 않습니다.

2. <Space>키의 선택 토글

선택을 원하는 항목에 위치한 후 <Space>키를 누르면 해당 항목이 선택됩니다. 이미 선택되어 있는 경우에는 선택이 해제됩니다.

3. <Ctrl-U> 선택 해제

선택된 모든 항목들의 선택을 해제합니다. 만일 선택된 항목이 하나도 없다면 전체 선택을 합니다.

4. <Ctrl-A> 전체 선택

현재 선택 상황에 관계없이 모든 항목을 선택합니다.

* 위의 방법으로 선택과 해제를 한 경우 화살표 키를 사용하여 목록을 이동해도 선택 상태는 변하지 않습니다. 그렇기 때문에 선택을
해제하려면 <Ctrl-U>키나 <새로 고침> 기능을 사용해야 합니다. 이는 선택한 파일을 대상으로 독서나 복사 등의 작업을 실행한 이후에도 동일합니다. 그러나 삭제나 이동처럼 작업 실행 이후 목록이 변하는 상황에서는 선택이 해제됩니다.


ⅩⅣ-6-4. 파일 처리와 팝업메뉴


센스 독서기에서는 윈도우즈 탐색기와 동일한 방식으로 폴더 및 파일의 복사, 이동, 삭제, 이름 변경 기능을 지원합니다. 기능키는 다음과 같습니다.

센스 탐색창 기능표
기능 기능키
클립보드로 복사 Ctrl-C
잘라내기 Ctrl-X
붙여넣기 Ctrl-V
삭제 Del
이름 변경 F2

파일 관리 기능과 더불어 센스 독서기에서는 윈도우 탐색기와 마찬가지로 팝업 메뉴 기능을 지원합니다. 원하는 항목에 위치한 후 <Popup>키를 누르면 팝업 메뉴가 호출되며, 사용법은 윈도우 탐색기와 동일합니다.

* 센스 독서기의 폴더 및 파일 관리 기능과 팝업 메뉴 기능은 모두 윈도우 탐색기와 호환됩니다. 즉, 탐색기에서 복사/잘라내기 기능을 실행한 후 독서기에서 붙여넣기를 해서 사용할 수도 있습니다.


ⅩⅣ-6-5. 정보 알림 기능들


센스 독서기에서는 다음과 같은 폴더 및 파일의 정보를 확인할 수 있습니다.

1. 폴더 정보: <Ctrl-K>

현재 폴더의 경로, 하위 폴더의 수, 파일의 수 등의 정보를 읽어줍니다.

2. 파일 정보: <Ctrl-L>

현재 포커스가 위치한 파일의 크기와 생성 날짜 등의 정보를 출력합니다.
