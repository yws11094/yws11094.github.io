ⅩⅣ-5-13-3. 오디오 변환 설정


현재 열려져 있는 문서 파일의 내용을 오디오 파일로 변환할 때, 적용될 파일 형식과 문서 기준을 설정합니다.

1. 파일 포멧
문서 파일의 오디오 변환은 MP3과 WAV 형식이 지원되며, 다음의 절차에 따라 설정을 진행합니다.
① 변환을 원하는 문서 파일을 독서창을 이용해 열고, 재생이 시작되면 <Enter> 키를 눌러 연속읽기를 중지합니다.
② <Alt> 키를 눌러 주메뉴를 열고 [설정] -> [오디오 변환] -> [파일 포멧] 메뉴를 실행합니다.
③ '오디오 변환 설정' 대화상자가 출력되면 처음에 위치한 목록에서 파일 형식을 선택합니다. 변환 가능한 파일 형식은 MP3와 WAV 파일입니다.
④ 파일 형식을 선택한 이후, <Tab> 키를 눌러 Sample Rate를 지정하는 목록상자로 이동하여 원하는 값을 지정합니다. 텍스트 파일을 변환할 경우 기본으로 설정하면 샘플 레이트가 16000으로 지정됩니다.
⑤ 음질 설정 목록에서 원하는 음질을 선택합니다.
파일 형식을 MP3로 설정한 경우 음질을 32K에서 320K까지 선택할 수 있으며, WAV로 선택한 경우 384에서 768까지 지정할 수 있습니다. 음질을 좋게 선택하면 파일의 사이즈가 증가함으로 사용자의 MP3 플레이어 용량에 따라 적절한 음질을 선택합니다.
⑥ 채널을 선택하는 라디오 버튼으로 이동한 다음, 사용자가 원하는 채널을 선택합니다. 텍스트 파일 변환의 경우 기본은 모노입니다.
⑦ '확인' 버튼을 눌러 지금까지의 설정을 적용합니다. <ESC> 키나 '취소' 버튼을 누르면 설정값이 취소됩니다.

2. 문서 기준
오디오 파일 변환시 파일을 하나로 만들 것인지, 기준에 따라 여러 개로 분할하여 만들 것인지 설정합니다.
① 문서 파일을 독서창에서 열고, 재생이 시작되면 <Enter> 키를 눌러 연속읽기를 중지합니다.
② <Alt> 키를 눌러 주메뉴를 열고 [설정] -> [오디오 변환] -> [문서 기준] 메뉴를 차례대로 실행합니다.
③ '문서 기준 설정' 대화상자가 나타나면 방향키를 이용하여 분할 기준을 선택한 뒤, '확인' 버튼을 눌러 설정을 적용합니다.
사용자가 선택할 수 있는 기준은 다음의 네 가지가 있습니다.
- 라인: 한 파일의 내용을 지정한 라인 수만큼으로 제한합니다.
- 시간: 한 파일의 내용을 지정한 시간만큼으로 제한합니다. 시간은 분 단위입니다.
- 목차: 한 파일에 하나의 목차 분량만을 담습니다.
- 없음: 파일을 분할하지 않고 한 개의 파일로 생성합니다.
　
* 라인이나 시간 옵션을 선택한 경우 '값' 편집창이 활성화됩니다. 여기에 사용자가 원하는 라인수나 시간을 입력합니다.
* 분할된 파일은 순서에 따라 파일명 끝부분에 -001, -002, -003과 같이 숫자가 붙어 순서를 구분해 줍니다.
* 파일 분할 옵션을 선택한 경우 하단에 '파일 번호' 체크박스가 활성화됩니다. 이 옵션을 선택하게 되면 파일 순서가 내용 앞에 출력됩니다.

3. 줄 간격 시간 설정
파일 내용 중 라인이 바뀌는 위치에 일정한 간격을 조절합니다.
대화상자가 출력되면 '간격' 편집창에 사용자가 원하는 값을 입력합니다. 시간은 1/1000초입니다. 만약, 0.2초의 간격을 지정하고 싶다면 '200'을 입력합니다.


ⅩⅣ-5-13-4. 이동 줄 설정


독서기의 기능 중 '이동 줄 전으로'와 '이동 줄 다음으로' 기능 사용시 이동될 줄의 수를 지정합니다.

[설정] 메뉴에서 [이동 줄 설정] 메뉴를 실행하면 '이동 줄 설정' 대화상자가 나타납니다. 이곳에
설정하고자하는 값을 입력한 후 '확인' 버튼을 누르면 됩니다. 기본값은 25입니다.


ⅩⅣ-5-13-5. 파일 열기시 자동 재생


기본적으로 센스 독서기는 파일을 열었을 때 그 파일을 자동으로 재생합니다.

경우에 따라서는 이것이 불편할 수도 있는데 이 때에는 [설정] 메뉴의 [파일 열기시 자동 재생] 메뉴를 통해 해제할 수 있습니다. 이 메뉴는 체크메뉴이기 때문에 한 번 실행할 때마다 선택/해제 상태가 토글됩니다.

이기능이 해제되면 파일을 연 후 자동으로 재생하지 않습니다. 이때에는 재생 기능을 사용하여 수동으로 재생할 수 있습니다.


ⅩⅣ-5-13-6. 음성 메시지 및 사운드 메시지 알림 설정


독서기 사용시 사용자의 편의를 위해 출력되는 음성 메세지와 사운드 메세지의 출력을 설정합니다.

1. 음성 메세지

독서 중 사용자가 어떤 기능을 실행하면 '1줄 다음으로', '다음 페이지로' 등과 같이 음성으로 상황 설명을 해 주는 것을 음성 메세지라 합니다.

설정 변경을 위해서는 [설정] 메뉴의 [음성 메시지 알림] 메뉴를 사용합니다. 체크 메뉴이기 때문에 실행시마다 선택이 토글됩니다.

편집 모드가 아닌 상태에서는 음성 메시지 알림을 <v>키나 <플러스(+)>키를 사용해서도 토글할 수 있습니다.

2. 사운드 메세지

독서 중 사용자의 기능 조작에 대해 효과음을 출력해 주는 것을 사운드 메세지라 합니다.

설정을 위해서는 [설정] 메뉴의 [사운드 메시지 알림] 메뉴를 사용합니다.

 사운드 메시지 알림 기능도 편집 모드가 아닌 상태에서는 <대쉬(-)>키를 사용해서 토글할 수 있습니다.


ⅩⅣ-5-13-7. 프로그램 창 활성 설정


독서창이 열려 있지 않은 상태에서 독서기로 파일을 열면 기본적으로 독서창이 열리면서 이 독서창이 활성화됩니다.

하지만 센스 독서기는 원격 제어가 가능하기 때문에 반드시 프로그램 창이 활성되어야 기능을 사용할 수 있는 것은 아닙니다. 오히려 활성되지 않는 것이 더욱 편리할 경우가 있는데 이럴 때 이 설정을 사용합니다.

[설정] 메뉴의 [프로그램 창 활성] 메뉴는 독서창의 활성 여부를 결정합니다. 체크 메뉴이기 때문에 실행시마다 선택과 해제가 토글됩니다.

만일 이 기능이 해제되어 있으면, 독서창이 열려도 바로 창이 활성되지는 않습니다. 이런 경우 원격 제어 기능을 사용하거나 독서창을 활성 상태로 변경하려면 <Alt-Tab>키를 눌러 활성 윈도우를 전환해 주어야 합니다.


ⅩⅣ-5-13-8. 자동 책갈피 설정


자동 책갈피를 설정하면 직접 책갈피를 저장하지 않아도 파일을 종료할 때 현재 라인이 자동적으로 책갈피로 저장됩니다.

자동 책갈피 설정은 [설정] 메뉴의 [자동 책갈피 설정] 메뉴를 사용합니다.

자동 책갈피 설정은 문서 파일과 미디어 파일을 별도로 지정할 수 있습니다. '자동 책갈피' 대화상자가 출력되면 문서 파일과 미디어 파일
체크상자를 각각 선택한 다음 '확인' 버튼을 눌러 설정을 적용합니다.


ⅩⅣ-5-13-9. 이퀄라이저 밴드 설정


기본적으로 설정된 5밴드의 이퀄라이저 외에 사용자의 취향에 맞도록 이퀄라이저를 설정할 수 있습니다.
[이퀄라이저 밴드 설정] 메뉴를 실행하면, 설정 대화상자가 출력됩니다. 대화상자에는 밴드 목록상자, 밴드 설정을 위한 '슬라이드, 삭제 및 추가' 버튼 등의 컨트롤들이 배치되어 있습니다.

'밴드 설정' 대화상자에 배치된 컨트롤의 세부 항목은 다음과 같습니다.
각 컨트롤의 이동은 <Tab> 키나 <Shift-Tab> 키를 사용합니다.

1. 밴드 목록상자
현재 설정되어 있는 밴드의 정보가 목록으로 표시됩니다.
삭제를 원하는 밴드가 있을 경우 해당 목록에 위치한 다음 '삭제' 버튼을 누릅니다.

2. 밴드 설정 슬라이드
새로운 밴드를 추가할 경우, 4개의 슬라이드를 원하는 값으로 설정하고 '추가' 버튼을 누릅니다. 각 슬라이드는 순서대로 주파수, 넓이, 최대값, 최소값을 지정합니다.
현재 설정된 밴드의 값을 변경하는 것은 불가능합니다.

3. 추가, 삭제 버튼
'추가' 버튼은 슬라이드에 지정된 값으로 새로운 밴드를 만들어 주며, '삭제' 버튼의 경우 밴드 목록에 선택된 밴드를 삭제합니다.

4. 리버브 사용 체크상자
이 옵션을 선택한 경우, 리버브 효과를 설정할 수 있는 밴드가 추가됩니다.
추가된 밴드는 목록에는 표시되지 않으며, <Ctrl-F10> 키를 눌러 밴드를 변경할 때 확인할 수 있습니다.


ⅩⅣ-5-13-10. 미디어 출력 설정


미디어 파일을 재생할 때에는 다음의 키를 사용하여 출력 크기와 속도를 조절할 수 있습니다.

독서기에서는 문서 파일과 미디어 파일의 음성 출력 설정을 각각 별도로 사용함에 유의합니다.

크기 감소: Pgdn

크기 증가: Pgup

속도 감소: Ctrl Pgdn

속도 증가: Ctrl Pgup

* 미디어 재생 속도는 일시적입니다. 종료후 다시 파일을 열었을 때에는 원래의 속도로 출력이 이루어집니다.


ⅩⅣ-5-13-11. 다음 파일 자동 재생


센스 독서기를 통해 여러개의 파일을 한꺼번에 열어서 재생하는 경우 이를 연속으로 재생할지 선택하는 메뉴입니다. 센스독서창 <설정> 메뉴에서 아래 방향키로 내려가서 "다음 파일 자동재생"을 <Enter>로 선택/해제 할 수 있습니다. 선택을 하면 현재 파일의 재생이 끝나면 다음파일을 자동으로 재생하고, 해제하면 현재 재생중인 파일만 재생하고 멈춥니다.

여러 파일을 한꺼번에 불러올 때는 센스 탐색창에서 스페이스바로 파일을 여러개 선택해서 재생하거나 윈도우즈 탐색기에서 여러 파일을 선택한 후 팝업키를 눌러서 센스 독서기로 재생 메뉴를 이용하세요.


ⅩⅣ-5-13-12. 원격 조정


독서기의 원격 제어 기능을 사용할 것인가를 선택합니다.
이 옵션이 선택되어 있으면 독서기의 원격 제어 기능이 활성화되고, 해제되어 있으면 원격제어 기능을 이용할 수 없습니다. 옵션의 선택과 해제는 해당 메뉴에서 <Enter> 키를 누르면 변경됩니다.


ⅩⅣ-5-13-13. 화면 확대/축소 <Ctrl-마우스 휠 위/아래>


화면의 글자를 확인할 수 있는 약시 사용자의 경우 <Ctrl> 키를 누른 상태에서 마우스 휠을 위/아래로 회전시키면 클라이언트 영역의 확대/축소가 가능합니다.
이 기능은 클라이언트 영역 내의 글자, 이미지, 테이블 등 모든 개체에 적용됩니다.
