ⅩⅣ-5-1. 독서창 - 독서모드, 편집모드


센스 독서기는 독서 모드와 편집 모드의 두 가지 사용 모드가 있습니다. 독서 모드와 편집 모드를 전환하려면 <Alt-Z>키를 누르면 됩니다. 단, 재생 중일 때는 두 모드를 전환할 수 없습니다.

1. 독서 모드

독서 모드는 파일을 불러왔을 때 기본적으로 동작하는 모드로 파일의 내용을 수정할 수 없습니다. 사용자가 임의로 센스 독서기의 설정을 변경하지 않았다면 독서 모드 상태에서 파일이 자동 재생됩니다.

2. 편집 모드

편집 모드는 글자의 입력, 수정, 삭제가 가능한 모드입니다. 일반적인 편집기와 동일하게 사용 가능합니다. 편집 모드는 문서 파일에서만 사용 가능하며 텍스트 파일로만 저장할 수 있습니다.


ⅩⅣ-5-2. 독서창 - 재생하기


1. 재생

재생은 센스 독서기에 의해 자동적으로 음성출력이 이루어지고 있는 상태를 뜻합니다. 기본적으로 센스 독서기는 파일을 열었을 때 자동으로 재생을 시작합니다. 따라서 별도의 조작 없이 독서가 바로 이루어지게 됩니다. 재생이 되지 않는 상태에서 재생을 하고자할 때에는 [재생] 메뉴에서 [재생] 메뉴를 사용하거나 <Ctrl-p>키를 사용합니다.
독서 모드 상태에서는 재생과 중지를 <Enter>키로 토글할 수 있습니다. 파일의 끝까지 재생했다면 자동으로 일시중지 상태로 들어가며, 여러 파일을 열어놓은 상태라면 자동으로 다음 파일로 넘어가 재생이 계속 됩니다.
만일, 자막 있는 동영상 파일을 재생한 상태라면 자막 내용을 음성으로 출력해 줍니다.

2. 정지

재생 중인 상태에서 재생 동작을 멈추려면 [재생] 메뉴의 [중지] 메뉴를 사용하거나 <Ctrl-Space>키를 사용합니다. 중지는 독서창 자체를 종료하는 것은 아니며 재생중이던 위치에서 재생 동작을 멈추는 것을 뜻합니다.
문서 파일의 경우 재생이 중지되면 수동으로 글자, 단어, 줄, 페이지 등을 이동하며 내용을 자세히 살펴볼 수 있습니다. 이때 사용되는 키와 기능은 일반 편집기와 동일합니다.
독서 모드 상태에서는 재생과 중지를 <Enter>키로 토글할 수 있습니다.

3. 일시 정지

재생 중에 잠시 독서를 멈추고자할 때 일시중지를 사용합니다. 일시중지를 사용하려면 [재생] 메뉴에서 [일시중지 및 재생] 메뉴를 사용합니다. 일시중지 상태에서 다시 한번 이 메뉴를 사용하면 재생이 시작됩니다. 일시중지와 재생은 <Space>키로 토글할 수 있습니다.
일시중지는 중지와는 다릅니다. 중지는 재생 자체를 끝내는 것이지만 일시중지는 재생 상태에서 잠시 기다리고 있는 것입니다. 그렇기 때문에 중지 상태에서 사용할 수 없는 5줄/10초 전, 5줄/10초 다음으로 이동, 50줄/5분전, 50줄/5분 다음으로 이동 등의 기능을 사용할 수 있습니다. 그러나 재생 상태에서 사용할 수 없는 블럭 지정 기능은 역시 일시중지 상태에서도 사용할 수 없습니다.

4. 구간 설정

미디어 파일을 재생 중 특정 부분을 반복해서 재생하고 싶을 때 구간설정 기능을 사용합니다. 구간 반복은 [재생] 메뉴에서 [구간 설정] 메뉴를 사용하거나 <Alt-L>키를 이용합니다. 미디어 파일을 재생하는 중 반복재생하고 싶은 시작 부분에서 <Alt-L>을 한 번 누르면 "구간지정"으로 음성출력하며 구간설정이 시작됩니다. 잠시 뒤 반복재생하고 싶은 부분 끝에서 <Alt-L>을 한 번 더 누르면 "구간지정완료"라고 음성출력하며 구간설정이 완료되고 지정된 구간을 반복해서 재생하기 시작합니다. 구간반복을 하는 중 <Alt-L>을 한 번 더 누르면 "구간지정해제"라고 음성출력하면서 구간설정을 해제하고 일반 상태로 재생을 시작합니다.


ⅩⅣ-5-3. 이퀄라이저 기능


오디오 파일 재생시 사용자의 취향에 따라 이퀄라이저로 음색을 조정할 수 있습니다.
이퀄라이저 기능은 사용자가 선택적으로 사용 가능하며, 기본적으로는 6밴드의 이퀄라이저가 제공되나, 밴드의 수와 음의 영역 모두 사용자가 개성에 맞게 설정해서 사용할 수 있습니다.

1. 이퀄라이저 사용/해제
오디오 파일 재생 중 <Ctrl-F9> 키를 눌러 이퀄라이저 사용/해제를 토글합니다.

2. 밴드의 지정
이퀄라이저를 선택한 이후 설정값을 변경할 밴드를 선택합니다. <Ctrl-F10> 키를 한 번씩 누를 때마다 밴드 1에서 6까지 차례대로 변경됩니다.

3. 밴드의 값 감소
변경을 원하는 밴드를 선택한 이후 <Ctrl-F11> 키를 누르면 밴드의 설정값이 1씩 감소합니다. 설정 가능한 값은 1부터 15까지이며 오프를 시키면 해당 영역은 원음과 동일하게 출력됩니다.

4. 밴드의 값 증가
밴드 선택 이후, <Ctrl-F12> 키를 누르면 밴드의 값이 1씩 증가됩니다.

5. 이퀄라이저 밴드 설정
센스 독서기에서 기본적으로 제공되는 6밴드 외에 사용자가 음의 영역과 밴드를 지정할 수 있습니다.


ⅩⅣ-5-4. 현재 줄, 단어, 글자 다시 읽기


문서를 읽는 도중 내용을 놓쳤거나 정확히 내용을 파악해야 할 경우 사용하는 기능입니다.

1. 재생 중 현재 라인 다시 읽기 <Ctrl-Enter>
문서가 자동 재생 중인 상태에서 <Ctrl-Enter> 키를 누르면 현재 라인을 다시 읽어줍니다.

2. 정지 상태에서 현재 줄, 단어, 글자의 확인
독서기가 문서를 재생하고 있는 상태에서 <Enter> 키를 누르면 재생이 정지됩니다. 정지 상태에서는 다음의 키들을 활용하여 현재 줄, 단어, 글자를 확인할 수 있습니다.
<Ctrl-,(컴마)> 현재 라인을 다시 읽어줍니다.
<Ctrl-.(마침표)> 현재 캐럿이 위치한 단어를 다시 읽어줍니다.
<Ctrl-/(슬레시)> 현재 글자를 다시 읽어줍니다. 반복해서 누를 경우 음소 단위로 풀어서 읽어줍니다.


ⅩⅣ-5-5. 정보 알림 기능들


센스 독서기에서는 다음과 같은 폴더 및 파일의 정보를 확인할 수 있습니다.

1. 폴더 정보: <Ctrl-K>

현재 폴더의 경로, 하위 폴더의 수, 파일의 수 등의 정보를 읽어줍니다.

2. 파일 정보: <Ctrl-L>

현재 포커스가 위치한 파일의 크기와 생성 날짜 등의 정보를 출력합니다.

ⅩⅣ-5-11. 오디오 파일 만들기 <F9>


현재 열려져 있는 문서 파일의 내용을 MP3이나 WAV 파일로 변환해 줍니다.
만들어진 오디오 파일은 독서기에서 재생하는 것과 똑같은 형태의 소리로 저장됩니다. 구두점을 뺀 상태라면 역시 오디오 파일에서도 구두점이 빠지며, 속도가 7인 경우에는 7의 속도로 변환이 됩니다. 이렇게 만들어진 오디오 파일은 휴대용 MP3 플레이어나 기타 재생기기를 사용해 언제 어디서나 독서기에서 독서를 하는 것처럼 간편하게 들을 수 있습니다.
이 기능은 음성엔진이 '보이스텍스트'로 설정되어 있는 경우에만 사용 가능합니다.
다음의 절차에 따라 문서 파일의 내용을 오디오 파일로 변환합니다.

① 변환을 원하는 문서 파일을 독서창으로 열고, <Enter>키를 눌러 연속읽기를 정지합니다.
② <F9> 키를 누르면 지정된 파일 포멧과 분할 기준에 따라 오디오 파일이 만들어 집니다.

* 오디오 파일의 포멧과 분할 기준은 [설정] -> [오디오 변환] 메뉴에서 미리 지정해야 합니다.
* 변환된 오디오 파일은 문서 파일이 위치한 폴더에 생성됩니다.


ⅩⅣ-5-12. 보조 음성 엔진 사용 토글 <Ctrl-E>


보조 음성 엔진의 동작을 자동모드/수동모드/해제 중에서 선택합니다.

- 자동모드: 출력 언어를 자동 판단하여 해당 엔진으로 출력합니다. 자동모드에서 사용할 엔진은 <보조음성엔진 선택> 대화상자를 이용하여 자동모드 사용 엔진으로 선택해 두어야 합니다. 예를 들어, <보조음성엔진 선택> 대화상자에서 영어엔진을 자동모드 사용 엔진으로 설정하였다면 한/영 혼용 문장에서 한글은 기본 음성엔진이 동작하고 영어는 영어 보조 음성엔진이 동작하게 됩니다. 해당 엔진이 없으면 기본엔진으로 출력됩니다.

- 수동모드: 출력엔진을 자동 변경하지 않고 <보조음성엔진 선택> 대화상자에서 지정된 엔진으로 출력합니다.

- 해제: 보조음성엔진을 사용하지 않습니다.

* 보조음성엔진의 속도, 높이, 크기 등의 설정을 변경하려면 <보조음성엔진 선택> 대화상자를 호출하여 값을 설정합니다.
