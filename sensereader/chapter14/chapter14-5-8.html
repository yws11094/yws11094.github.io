ⅩⅣ-5-8-1. 처음, 끝으로 이동 <Home>, <End>


파일 재생시 다음의 기능을 사용하여 파일의 처음과 끝으로 이동할 수 있습니다. 이 기능들은 정지시에는 동작하지 않습니다.

1. 처음으로 이동

[이동] 메뉴의 [처음으로] 메뉴를 사용하거나 <h> 혹은 <Home> 키를 누르면 현재 파일의 처음으로 이동하게 됩니다.

2. 끝으로 이동

[이동] 메뉴의 [끝으로] 메뉴를 사용하거나 알파벳 <e> 혹은 <End> 키를 누르면 끝으로 이동할 수 있습니다.

* 문서 파일의 경우 <Ctrl-Home>과 <Ctrl-End>키를 눌러 문서 파일의 처음과 마지막으로 이동할 수 있습니다.


ⅩⅣ-5-8-2. 페이지 이동 <PgUp>, <PgDn>


현재 파일이 문서 파일이라면 [이동] 메뉴의 [이전 페이지로] 메뉴를 사용하거나 <Pgup>키를
누르면 현재 위치에서 한 페이지 이전으로 이동합니다.

마찬가지로, [이동] 메뉴의 [다음 페이지로] 메뉴를 사용하거나 <Pgdn>키를 누르면 다음 페이지로 이동합니다.

재생 상태에서는 <p>, <n> 키가 페이지 이동 키로 사용이 가능합니다.

한 페이지는 현재 출력되는 화면의 크기입니다. 그렇기 때문에 출력되고 있는 창의 크기 상태에 따라 페이지 이동의 이동 줄수는 달라질 수 있습니다.

* VBF 파일의 경우 페이지에 대한 구분은 컨텐츠 제작시 삽입된 페이지 기호를 기준으로 합니다.


ⅩⅣ-5-8-3. 재생시에 사용되는 여러 이동 기능


다음의 기능들은 재생시에만 사용되는 이동 기능들입니다. 중지상태에서는 동작하지 않습니다.

각 기능은 [이동] 메뉴 아래 포함되어 있으며 단순히 원하는 기능에 해당하는 메뉴를 실행만 하면 동작합니다. 줄 이동은 문서 파일의 경우에, 시간(초, 분)이동은 미디어 파일 재생시에 해당합니다. 아래는 기능에 해당하는 키를 설명한 것입니다.

① 1줄(5초)이전, 다음으로: 위, 아래 화살표

② 5줄(10초) 이전, 다음으로: 좌, 우 화살표

③ 10줄(1분) 이전, 다음으로: Ctrl 위, Ctrl 아래 화살표

④ 50줄(5분) 이전, 다음으로: Ctrl 좌, Ctrl 우 화살표

⑤ 100줄(10분)이전, 다음으로: Alt 좌, Alt 우 화살표


ⅩⅣ-5-8-4. 이동줄 이동 <Ctrl-PgUp>, <Ctrl-PgDn>


문서 파일의 경우 [이동] 메뉴의 [이동줄 전으로] 메뉴를 사용하거나 <Ctrl-Pgup>키를
누르면 이동줄 설정에 의해 지정된 줄수만큼 이전 줄로 이동합니다.

마찬가지로, [이동] 메뉴의 [이동줄 다음으로] 메뉴를 사용하거나 <Ctrl-Pgdn>키를 누르면 이동줄만큼 다음으로 이동합니다.

이동줄의 줄 수는 사용자가 임의로 지정할 수 있으며, [설정] 메뉴의 [이동줄 설정] 메뉴를 사용해서 지정합니다.


ⅩⅣ-5-8-5. 단락간 이동 <Shift-F7>, <F7>


문서 파일에서는 현재 위치를 중심으로 이전, 다음 단락 단위의 이동을 할 수 있습니다.

이전 단락으로 이동을 원할 때에는 [이동] 메뉴의 [이전 단락으로 이동] 메뉴를 사용하거나 <Shift-F7>키를 사용합니다. 다음 단락으로 이동은 [이동] 메뉴의 [다음 단락으로 이동] 메뉴를 사용하거나 <F7>키를 사용하면 됩니다. 재생시에는 각각 <Shift-U>, <U> 키를 사용할 수도 있습니다.

한 단락의 구분은 [설정] 메뉴의 [단락 기준 설정]에서 설정한 기준값에 따라 달라지며 이 기준은 빈줄, 들여쓰기, 빈줄과 들여쓰기 중 하나가 될 수 있습니다. 기본값은 빈줄입니다.

* VBF 파일의 경우 단락 기준은 컨텐츠 제작시 삽입된 단락 기호를 기준으로 합니다.


ⅩⅣ-5-8-6. 마크간 이동 <Shift-F8> <F8>


현재 사용중인 파일에 마크가 지정되어 있으면 이전, 다음 마크로 이동할 수 있습니다.

이전 마크로의 이동은 [이동] 메뉴의 [이전 마크로] 메뉴를 사용하거나 <Shift-F8>키를 사용하고, 다음 마크로의 이동은 [이동] 메뉴의 [다음 마크로] 메뉴를 사용하거나 <F8>키를 사용합니다. 이때 단축마크로는 이동하지 않습니다. 재생시에는 각각 <Shift-I>, <I> 키를 사용할 수도 있습니다.

마크를 지정하고자할 때에는 [고급] 메뉴의 [마크] 메뉴에서 [마크지정] 메뉴를 사용합니다.


ⅩⅣ-5-8-7. 목차간 이동 <Shift-F2> <F2>


목차가 지정된 문서 파일에서 이전, 다음 목차로 이동합니다.

이전 목차로의 이동은 [이동] 메뉴의 [이전 목차로] 메뉴를 사용하거나 <Shift-F2>키를 사용하고, 다음 목차로의 이동은 [이동] 메뉴의 [다음 목차로] 메뉴를 사용하거나 <F2>키를 사용합니다.


ⅩⅣ-5-8-8. 특정 위치로 이동 <Ctrl-G>


사용 중인 파일의 특정 위치로 이동을 원할 때에는 "특정 위치로 이동" 기능을 사용합니다. 이 기능을 사용하기 위해서는 [이동] 메뉴의 [특정 줄로 이동] 메뉴를 실행하거나 <Ctrl-g>키를 누릅니다. 재생시에는 <G>키를 사용해도 됩니다.

기능을 실행하면 '위치 입력' 대화상자가 나타나며 '위치' 편집창이 포커스됩니다. 이곳에 원하는 값을 입력하고 '확인' 버튼을 눌러주면 입력된 수치에 따라 이동이 이루어집니다.

입력 규칙은 파일 종류에 따라 다음과 같습니다.

1. 문서 파일에서의 이동

① 입력된 수는 줄 번호로 간주합니다.

② 아무런 기호 없이 숫자만 입력되면 해당 줄로 이동합니다.

③ 숫자 이전에 대쉬(-) 기호가 입력되면 현재 줄을
중심으로 입력한 수치만큼 이전 줄로 이동합니다.

④ 숫자 이전에 플러스(+) 기호가 입력되면 현재 줄을
중심으로 입력된 수치만큼 다음 줄로 이동합니다.

2. 미디어 파일에서의 이동

① 이동할 시간을 입력하돼 "시:분:초"의 형태를
따릅니다.

② 0은 입력하지 않아도 됩니다. 예를 들어 "1:0:30"은 "1::30"으로 입력하면 됩니다.

③ 앞쪽부터 자리의 생략이 가능하며 생략된 자리는 0으로 취급합니다. 예를 들어 "30"은 "0:0:30"과 같고 "1:30"은 "0:1:30"과 같습니다.

④ 단순히 시간만을 입력하면 해당 시간 위치로 이동합니다.

⑤ 시간 앞에 대쉬(-) 기호가 입력되면 현재 위치에서 시간만큼 앞으로 이동되고 플러스(+) 기호가 입력되면 시간만큼 뒤로 이동합니다.

* VBF 파일의 경우 '위치' 편집창에 'p' 기호를 삽입한 다음 숫자를 입력하면 해당되는 페이지로 바로 이동할 수 있습니다.

예: 'p20'라고 입력한 경우 20페이지로 이동됨.


ⅩⅣ-5-8-9. 특정 객체로 이동 <Shift-Tab>, <Tab>


객체란 문서 내에 삽입된 이미지, 링크, 주석, 멀티미디어 데이터 등을 의미합니다. 객체 이동은 모든 파일 형식에서 지원되는 것은 아니고 객체가 삽입된 VBF 2.0 파일에서만 사용 가능합니다.

1. 객체 이동과 목록보기

재생과 중지 상태에서 다음 객체로 이동하려면 <Tab> 키를 누르고 이전 객체로 이동하려면 <Shift-Tab> 키를 누릅니다.

또한, 문서 내에 삽입된 객체를 목록으로 출력하여 내용을 확인하거나 이동하려면 [이동] 메뉴의 [특정 객체로 이동] 메뉴를 실행하거나 <Alt-O> 키를 누릅니다.

'객체 목록' 대화상자는 다음 3개의 목록으로 구성되고 각 컨트롤의 이동은 <Tab> 키나 <Shift-Tab> 키를 사용합니다.

1) 범위 선택 라디오 버튼

문서 전체와 현재 목차 중 하나의 옵션을 선택합니다. 전체를 선택할 경우 문서 전체의 객체를 목록에 표시하고 목차를 선택할 경우 현재 목차 내에 삽입된 객체만을 표시합니다.

2) 종류 선택 라디오 버튼

객체의 구분을 위해 이미지, 링크, 주석 등 3개의 옵션이 있습니다. 링크에는 미디어 객체도 포함됩니다.

3) 객체 목록상자

위의 컨트롤에서 범위와 종류를 선택한 다음 객체 목록상자로 이동하면 해당 조건에 맞는 객체가 목록으로 표시됩니다. 위/아래 화살표 키를 이용하여 내용을 확인할 수 있으며, 해당 객체로 바로 이동하려면 <Enter> 키를 누릅니다.

2. 링크 객체의 이동과 미디어 객체의 재생

미디어 객체를 포함한 링크 객체를 실행하기 위해서는 해당 객체로 이동한 다음 <Ctrl-Enter> 키를 누릅니다.
일반적인 링크 객체를 실행했다면 링크와 연결된 홈페이지나 문서가 자동으로 열리게 되고, 미디어 객체를 실행했다면 동영상 재생기가 자동으로 실행되어 미디어 내용을 재생해 줍니다.

미디어 재생기에서 사용 가능한 기능키는 다음과 같습니다.

Esc : 미디어 재생을 정지하고 이전 문서로 돌아갑니다.
위/아래 화살표키 : 미디어의 볼륨을 조절합니다.
좌/우 화살표키 : 3초 이전, 3초 다음으로 이동합니다.


ⅩⅣ-5-8-10. 찾기 기능


문자열을 찾기 위해서는 [이동] 메뉴의 [찾기] 메뉴를 사용하거나 <Ctrl-f>키를 누릅니다. 재생 상태에서는 <W>키를 사용해도 됩니다.

'문자열 찾기' 대화상자가 나타나면 찾기를 원하는 문자열을 입력하고 원하는 옵션을 선택한 후 '확인' 버튼을 누릅니다.

문자열 찾기에서 사용할 수 있는 옵션은 다음과 같습니다.

1. 검색 방향 라디오 버튼: 현재 위치에서 찾기를 수행할 방향을 선택하며 아래로 검색, 위로 검색 중 하나를 선택합니다. 기본값은 아래로 검색입니다.

2. 대소문자 구별 체크상자: 영문 대문자와 소문자를 구별하여 찾기를 수행할지 선택/해제합니다. 기본값은 해제입니다.

찾기에 사용된 문자열은 최근 20개까지 기억됩니다. 그렇기 때문에 기존에 입력했던 문자열을 다시 찾으려 할 때 검색 문자열을 다시 입력할 필요 없이 편집창에서 위, 아래 화살표로 원하는 문자열을 찾아볼 수도 있습니다.

[이동] 메뉴의 [계속 찾기] 메뉴를 수행하거나 <f3>키를 누르면 방금 전 '찾기' 대화상자를 통해 입력한 문자열과 옵션으로 다시 찾기를 하므로 원하는 문자열이 나올 때까지 계속하여 문자열을 찾아볼 수 있습니다.

또한 [이동] 메뉴의 [반대로 찾기] 메뉴를 사용하거나 <Shift-f3>키를 누르면 이전 찾기에서 방향만 반대로하여 계속 문자열을 찾아볼 수도 있습니다.


ⅩⅣ-5-8-11. 책갈피로 이동 <Ctrl-Q>


[이동]메뉴의 [책갈피로 이동]메뉴를 실행하거나 <Ctrl-Q>키를 누르면 저장된 책갈피 위치로 이동합니다.
