ⅩⅣ-5-7-1. 블럭 지정 <Shift-방향키> <Alt-L>


현재 파일의 일부 내용을 다른 프로그램에 옮기거나 파일로 저장하려 할 때 블럭 기능을 사용합니다.

블럭 기능을 사용하기 위해서는 먼저 파일 내용 중 사용하려는 부분을 블럭으로 지정해야 합니다. 센스 독서기에서는 윈도우즈의 표준적인 블럭 설정 방법과 함께 단축키를 이용한 블럭 설정 방법을 제공합니다.

다음의 방법 중 한 가지를 사용하여 블럭을 지정할 수 있습니다.

1. 윈도우즈의 표준적인 블럭 설정

① 블럭을 지정할 처음 위치로 이동합니다.

② <Shift>키를 누른 상태에서 화살표 키나 편집키를 이용하여 블럭 지정을 원하는 마지막 부분까지 이동합니다.

블럭 지정 후 화살표키나 편집키를 눌러 다른 곳으로 이동하면 블럭이 해제되므로 바로 블럭 기능을 사용해야 합니다.

2. 단축키를 이용한 블럭 설정

① 블럭 지정을 원하는 처음 위치의 글자로 이동 후 <Alt-L>키를 누릅니다.

② 블럭 지정을 원하는 마지막 글자로 이동 후 다시 <Alt-L>키를 누르면 그만큼의 내용이 블럭으로 선택됩니다.

이렇게 블럭을 지정한 상태에서는 다른 곳으로 이동해도 블럭이 해제되지 않습니다.

3. 문서 전체의 블럭 설정

현재 열려진 문서 전체를 블럭으로 설정하려면 <Ctrl-a>키를 누릅니다.

문서 전체를 블럭으로 설정한 이후 화살표키나 방향키를 눌러 캐럿의 위치가 이동되면 설정한 블럭이 해제된다는 점에 유의합니다.

　* 블럭 지정을 포함하여 모든 블럭 기능은 재생 상태에서는 사용할 수 없음으로 반드시 <Enter> 키를 눌러 재생을 중지한 상태에서 이용하여야 합니다.


ⅩⅣ-5-7-2. 실행 취소 <Ctrl-Z>


실행 취소는 편집 모드 상태에서 바로 이전에 실행했던 작업을 취소할 때 사용합니다. 삭제나 입력, 붙여 넣기 등의 작업을 할 때 실수가 있었다면 이 기능을 이용하여 원래의 상태로 되돌릴 수 있습니다.


ⅩⅣ-5-7-3. 클립보드 블럭 기능


센스 독서기의 편집창에서는 문서의 편집 작업을 위해 다음과 같은 블럭 편집 기능을 제공합니다.

1. 잘라내기 <Ctrl-X>

현재 블럭으로 선택되어 있는 부분을 잘라내어 클립보드에 넣습니다. 이 기능은 편집 모드에서만 동작합니다.

2. 복사 <Ctrl-C>

블럭으로 지정되어 있는 부분을 복사하여 클립보드에 넣습니다. 이때 클립보드에 있던 내용은 사라집니다.

3. 이어복사 <Ctrl-R>

블럭으로 지정되어 있는 부분을 클립보드에 덧붙입니다. 복사와는 달리 이전 클립보드의 내용은 사라지지 않습니다.

4. 붙여넣기 <Ctrl-V>

클립보드에 있는 내용을 현재 위치에 붙여 넣습니다. 이 기능은 편집 모드에서만 동작합니다.

* VBF 파일의 경우 컨텐츠 보호를 위해 클립보드 블럭 기능은 동작하지 않습니다.


ⅩⅣ-5-7-4. 파일 블럭 기능


센스 독서기에서는 편집기능의 향상을 위해 클립보드 블럭 기능과 함께 파일 블럭 기능을 제공합니다.

파일 블럭 기능을 사용하면 문서 내에서 필요한 부분을 직접 파일로 저장하거나 붙여넣기를 할 수 있습니다.

1. 파일로 잘라내기 <Alt-X>

블럭의 내용을 잘라내어 지정한 캡쳐 파일에 넣습니다. 이 때, 이미 캡쳐 파일이 존재하고 있었다면 파일을 덮어씁니다. 이 기능은 편집 모드에서만 동작합니다.

캡쳐 파일의 이름은 사용자가 지정할 수도 있으며, 센스리더 설치시 기본적으로 'xvbrd.cap' 파일로 지정되어 있습니다.

2. 파일로 복사 <Alt-C>

블럭의 내용을 지정한 캡쳐 파일에 복사합니다.

이때 이미 파일이 존재하고 있었다면 파일을 덮어씁니다.

3. 파일로 이어복사 <Alt-R>

블럭의 내용을 지정한 캡쳐파일에 덧붙입니다.

"파일로 복사" 기능과는 달리 이미 파일이 존재하고 있을 경우 그 파일 뒤에 내용을 추가합니다.

4. 끼워넣기 <Alt-V>

지정한 파일의 내용을 현재 위치에 붙여넣습니다.

끼워넣기 기능을 실행하면 '파일 선택' 대화상자가 나타나고 이 곳에서 내용을 가져올 파일을 지정한 후 '열기' 버튼을 누르면 그 파일의 내용이 현재 캐럿 위치에 들어옵니다. 이 기능은 편집 모드에서만 동작합니다.

* VBF 파일의 경우 컨텐츠 보호를 위해 파일 블럭 기능은 동작하지 않습니다.


ⅩⅣ-5-7-5. 삭제 기능


편집 모드에서 사용할 수 있는 삭제 기능은 크게 다음 세 가지가 있습니다.

1. 삭제: 블럭으로 지정한 내용을 모두 삭제합니다. 블럭 지정 후 [편집] 메뉴의 [삭제] 메뉴를
실행하거나 <Del>키를 누르면 블럭으로 설정된 부분이 삭제됩니다.

2. 현재 줄 삭제: 현재 캐럿이 있는 위치의 줄을 삭제합니다. [편집] 메뉴의 [현재 줄 삭제] 메뉴를
실행하거나 <Ctrl-Y>키를 누릅니다.

3. 빈 줄 삭제: 현재 문서 중에 있는 모든 빈 줄을 삭제합니다. [편집] 메뉴의 [빈 줄 삭제] 메뉴를
실행합니다.


ⅩⅣ-5-7-6. 문자열 바꾸기<Ctrl-H>


편집 모드 상태에서 사용자가 검색한 문자열을 지정한 문자열로 변환해 줍니다.

다음의 과정에 따라 문자열 바꾸기를 실행합니다.

① [편집] 메뉴의 [문자열 바꾸기] 메뉴를 실행하거나 기능키를 누릅니다.

② '바꾸기' 대화상자가 출력되면 '검색 문자열' 편집창에 찾고자 하는 문자열을 입력합니다.

③ '바꿀 문자열' 편집창에 바꿀 문자열을 입력합니다.

④ 검색 방향과 대소문자 구별을 원하는 방식으로 선택합니다.

⑤ '확인' 버튼을 누릅니다.

⑥ '문자열 확인' 대화상자가 나타나면 내용을 확인하고 바꾸기를 원하면 '예' 버튼을,
건너뛰기를 원하면 '아니오' 버튼을 누릅니다. 만일 질문 없이 모든 내용을 바꾸고자 할 경우 '모두 예' 버튼을 누릅니다.
