윈도우 8 이상 환경에서 터치 제스처를 통해 센스리더의 기능을 이용할 수 있습니다.
단 모니터가 터치가 지원되어야 아래의 기능을 이용하실 수 있으며, 윈도 8 이하 운영체제에서는 동작하지 않습니다.
제스처에 대한 기능은 아래와 같습니다.

1. 객체 관련

- 한 손가락 탭 또는 마우스나 손가락 대고 이동 = 현재 위치의 객체로 이동
- 한 손가락 왼쪽 쓸기 = 이전 객체 이동
- 한 손가락 오른쪽 쓸기 = 다음 객체 이동
- 한 손가락 위로 쓸기 = 상위 객체 이동
- 한 손가락 아래로 쓸기 = 하위 객체 이동
- 세 손가락 두 번 탭 = 이동한 객체 포커스
- 네 손가락 두 번 탭 = 하위 객체 자동 탐색 토글

- 한 손가락 두 번 탭 또는 한 손가락 대고 다른 한 손가락 탭 = 현재 객체 실행
- 한 손가락 세 번 탭 또는 한 손가락 대고 다른 한 손가락 두 번 탭 = 팝업 메뉴 호출

2. 방향키 관련

- 두 손가락 상 하 좌 우 쓸기 = 상 하 좌 우 방향키
- 한 손가락 대고 다른 한 손가락 상 하 좌 우 쓸기 = 상 하 좌 우 방향키
- 한 손가락 대고 두 손가락 상 하 좌 우 쓸기 = 컨트롤 상 하 좌 우 방향키

- 세 손가락 좌 우 쓸기 = 홈 엔드 키
- 세 손가락 상 하 쓸기 = 페이지업 페이지다운 키
- 한 손가락 대고 세 손가락 좌 우 쓸기 = 컨트롤 홈 엔드 키
- 한 손가락 대고 세 손가락 상 하 쓸기 = 컨트롤 페이지업 페이지다운 키

* 한 손가락 대는 동작은 양손을 사용하면 더 편리합니다.

3. 탭 엔터 이스케이프 관련

- 두 손가락 두 번 탭 또는 한 손가락 대고 두 손가락 탭 = 엔터 키
- 두 손가락 세 번 탭 또는 한 손가락 대고 두 손가락 두 번 탭 = 백스페이스 키
- 네 손가락 위로 쓸기 또는 한 손가락 대고 세 손가락 탭 = 이스케이프 키
- 네 손가락 아래로 쓸기 또는 한 손가락 대고 세 손가락 두 번 탭 = 알트 F4 키

- 네 손가락 왼쪽 쓸기 = 쉬프트 탭 키
- 네 손가락 오른쪽 쓸기 = 탭 키
- 한 손가락 대고 네 손가락 왼쪽 쓸기 = 알트 쉬프트 탭 키
- 한 손가락 대고 네 손가락 오른쪽 쓸기 = 알트 탭 키

4. 그밖에 명령

- 두 손가락 탭 = 음성 멈추기
- 두 손가락 네 번 탭 = 마지막 내용 클립보드에 복사
- 세 손가락 세 번 탭 = 출력 내용 호출
