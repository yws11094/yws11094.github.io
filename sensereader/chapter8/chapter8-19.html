Ⅷ-19. 위치 정보 관련 기능

  • Ⅷ-19-2. 글자 코드 알림 <없음>

  • Ⅷ-19-5. 캐럿 위치 글자 코드 알림 <없음>
  • Ⅷ-19-6. 캐럿 위치 글자 속성 알림 <Ctrl-Insert-A>

Ⅷ-19-1. 좌표 알림 <Insert-C>


현재 읽기 포인터가 위치한 좌표를 '화면좌표 x, y 윈도우좌표 x, y' 형식으로
읽어줍니다.


Ⅷ-19-2. 글자 코드 알림 <없음>


현재 읽기포인터가 위치한 글자의 코드를 WideCharacter(유니코드) 값으로 알려줍니다.

글자가 없으면 비프음을 냅니다.

예: 'A' '65'


Ⅷ-19-3. 글자 속성 알림 <Insert-A>


현재 읽기포인터가 위치한 글자의 속성을 알려줍니다.

글자 속성 알림 기능을 한 번 누르면 폰트명, 글자 포인트, 속성(언더라인, 기울임, 진하게) 등의 속성을 알려주고, 다시 한 번 더 누르면 배경색과 글자색을 RGB 형식으로 읽어줍니다.

예: '바탕 12포인트 언더라인' '배경 R 0 G 0 B 0 글자 R 255 G 255 B 255'

# RGB란 RED, GREEN, BLUE의 빛의 삼원색 조합을 의미하며, 윈도우즈에서는 각각의 색의 농도를 256 단계로 구별하여 색상을 표현하게 됩니다. RGB 값이 모두 0이면 검정색을 의미하고, RGB 값이 모두 256이면 흰색입니다.


Ⅷ-19-4. 캐럿 좌표 알림 <Ctrl-Insert-C>


현재 캐럿이 위치한 좌표를 '화면좌표 x, y 윈도우좌표 x, y' 형식으로 읽어줍니다.


Ⅷ-19-5. 캐럿 위치 글자 코드 알림 <없음>


현재 캐럿이 위치한 글자의 코드를 WideCharacter(유니코드) 값으로 읽어줍니다.


Ⅷ-19-6. 캐럿 위치 글자 속성 알림 <Ctrl-Insert-A>


현재 캐럿이 위치한 글자의 속성을 알려줍니다.

한 번 누르면 폰트명, 폰트 크기, 글자의 속성 상태등을 읽어주고 두 번 누르면 RGB 형태으로 글자의 배경색과 글자색을 알려줍니다.
