Ⅷ-1-1. 글자 단위 읽기


이전 글자로 <Numpad4> <Alt-Shift-M>
현재 글자 읽기 <Alt-Numpad5> <Alt-Shift-,>
다음 글자로 <Numpad6> <Alt-Shift-.>

읽기 포인터를 기준으로 글자단위로 해당 글자를 읽어주는 방법입니다.
글자 단위 읽기는 [음성 세부설정]의 구두점, 특수문자 읽기 설정과는 관계없이 모두 읽어주게 됩니다. 이는 글자 단위 읽기 자체가 세부적인 정보를 얻기 위한 목적으로 사용하는 빈도가 높기 때문입니다.

1. 이전 글자로 <Numpad4> <Alt-Shift-M>

현재 읽기 포인터의 이전 글자로 읽기 포인터를 이동하면서 해당 글자를 읽어줍니다.
읽기 포인터의 위치가 줄의 처음이면 비프음을 내고 이전 줄의 마지막 글자로 읽기 포인터를 이동하면서 글자를 읽어줍니다. 더 이상 이전 글자가 없으면 비프음을 출력합니다.

2. 현재 글자 읽기 <Alt-Numpad5> <Alt-Shift-,>

현재 읽기 포인터가 위치한 글자를 읽어주며, 글자가 아닌 경우 비프음을 출력합니다.
현재 글자를 자세히 풀어서 읽으려면 '현재 글자 읽기' 기능을 연속해서 실행합니다.

3. 다음 글자로 <Numpad6> <Alt-Shift-.>

현재 읽기 포인터의 다음 글자를 읽은 후에 읽기 포인터를 글자 위치로 이동시킵니다.
읽기 포인터가 줄의 끝이면 비프음을 내고 다음 줄의 처음 글자를 읽습니다. 더이상 다음 글자가 없으면 비프음을 냅니다.


Ⅷ-1-2. 단어 단위 읽기


이전 단어로 <Ctrl-Numpad4> <Insert-Numpad4> <Alt-Shift-J>
현재 단어 읽기 <Ctrl-Numpad5> <Insert-Numpad5> <Alt-Shift-K>
다음 단어로 <Ctrl-Numpad6> <Insert-Numpad6> <Alt-Shift-L>

1. 이전 단어로 <Ctrl-Numpad4> <Insert-Numpad4> <Alt-Shift-J>

현재 읽기 포인터의 이전 단어를 읽은 후에 단어의 첫글자로 읽기 포인터를 이동합니다.
읽기 포인터가 줄의 처음이면 비프음을 내고 이전 줄의 마지막 단어를 읽으면서 읽기 포인터가 이동합니다. 더 이상 이전 단어가 없는 경우에는 비프음을 출력합니다.

2. 현재 단어 읽기 <Ctrl-Numpad5> <Insert-Numpad5> <Alt-Shift-K>

읽기 포인터가 위치한 현재 단어를 읽습니다.
단어를 자세히 읽으려면 '현재 단어 읽기' 기능을 연속해서 누릅니다.

3. 다음 단어로 <Ctrl-Numpad5> <Insert-Numpad5> <Alt-Shift-L>

현재 읽기 포인터의 다음 단어를 읽은 후에 단어의 첫 글자로 읽기 포인터를 위치시킵니다.
읽기 포인터가 줄의 마지막이면 비프음을 내고 다음 줄의 처음 단어를 읽습니다. 다음 단어가 더이상 없을 경우에는 비프음을 출력합니다.


Ⅷ-1-3. 줄 단위 읽기


이전 줄로 <Numpad8> <Alt-Shift-U>
현재 줄 읽기 <Numpad5> <Ctrl-Shift-I>
다음 줄로 <Numpad2> <Alt-Shift-O>
윈도우 첫 줄로 <Ctrl-Numpad8> <Alt-Shift-P> <Insert-Numpad
위쪽 화살표>
윈도우 끝 줄로 <Ctrl-Numpad2> <Alt-Shift-;> <Insert-Numpad
아래쪽 화살표>

1. 이전 줄로 <Numpad8> <Alt-Shift-U>

현재 읽기 포인터가 위치한 이전 줄을 읽고 읽기 포인터를 이동합니다. 이전 줄이 없을 경우에는 비프음을 출력합니다. 이전 줄을 읽을 때는 '이동줄 읽기 설정'에서 설정한 방법대로 읽게 됩니다.

2. 현재 줄 읽기 <Numpad5> <Ctrl-Shift-I>

현재 읽기 포인터가 위치한 라인을 읽습니다. 현재 줄이 없으면 비프음을 냅니다.

3. 다음 줄로 <Numpad2> <Alt-Shift-O>

현재 읽기 포인터가 위치한 다음 줄을 읽고 다음 줄로 읽기 포인터를 이동합니다. 다음줄이 없을 때는 비프음을 출력합니다. 다음 줄을 읽을 때는 '이동줄 읽기 설정'에서 설정한 방법대로 읽게 됩니다.

4. 윈도우 첫 줄로 <Ctrl-Numpad8> <Alt-Shift-P> <Insert-Numpad
위쪽 화살표>

현재 지정된 윈도우 범위의 처음 줄을 읽고 읽기 포인터를 첫 줄로 이동시킵니다. 내용이 없으면 비프음을 냅니다.

5. 윈도우 끝 줄로 <Ctrl-Numpad2> <Alt-Shift-;> <Insert-Numpad2>

현재 지정된 윈도우 범위의 마지막 줄을 읽고 읽기 포인터를 해당 줄로 이동시킵니다. 내용이 없으면 비프음을 출력합니다.

참고사항 : 이동줄 읽기 방법의 예

"여러분은 지금 센스리더의 도움말을 보고 계십니다."라는 문자열이 있고, 현재 읽기 포인터가 '센' 글자 위치에 있다고 가정합니다.

1. 전체 읽기 : "여러분은 지금 센스리더의 도움말을 보고 계십니다."라고 라인 전체를 읽습니다.

2. 왼쪽 읽기 : "여러분은 지금"라고 음성출력을 합니다.

3. 오른쪽 읽기 : "센스리더의 도움말을 보고 계십니다."라고 읽습니다.

4. 단어 읽기 : "센스"라고 읽기 포인터가 위치한 단어만 읽습니다.


Ⅷ-1-4. 줄 처음/끝 읽기


줄 처음으로 <Numpad Home> <Alt-Shift-N>
줄 끝으로 <Numpad End> <Alt-Shift-/>

1. 줄 처음으로 <Numpad Home> <Alt-Shift-N>

현재 읽기 포인터가 위치한 줄의 처음 글자를 읽고 읽기 포인터를 줄 처음으로
이동합니다. 글자가 없을 경우에는 비프음을 냅니다.

2. 줄 끝으로 <Numpad End> <Alt-Shift-/>

현재 읽기 포인터가 위치한 줄의 마지막 글자를 읽고 읽기 포인터를 해당 글자로 위치시킵니다. 글자가 없을 경우에는 비프음을 냅니다.


Ⅷ-1-5. 이동 줄 단위 읽기


이동 줄 위로 <Numpad9> <Alt-Shift-Y>
이동 줄 아래로 <Numpad3> <Alt-Shift-H>

1. 이동 줄 위로 <Numpad9> <Alt-Shift-Y>

현재 읽기 포인터가 위치한 줄에서 이동줄 설정값만큼 라인을 위로 이동하여 읽고 읽기 포인터를
위치시킵니다. 만일 현재 윈도우 범위의 이전 줄이 이동줄 설정의 줄 수보다 적게 남아 있으면 윈도우 첫줄을 읽게
됩니다.

2. 이동 줄 아래로 <Numpad3> <Alt-Shift-H>

현재 읽기 포인터가 위치한 줄에서 이동줄 설정값만큼 라인을 아래로 이동하여 읽고 읽기 포인터를
위치시킵니다. 만일 현재 윈도우 범위의 아래 줄이 이동줄 설정의 줄 수보다 적게 남아 있으면 윈도우 첫 줄을 읽게
됩니다.


Ⅷ-1-6. 객체 단위 읽기


이전 객체로 <CTRL-Insert-Numpad4> <CTRL-Insert-왼쪽 화살표>
현재 객체 읽기 <CTRL-Insert-Numpad5>
다음 객체로 <CTRL-Insert-Numpad6> <CTRL-Insert-오른쪽 화살표>

1. 이전 객체로 <CTRL-Insert-Numpad4> <CTRL-Insert-왼쪽 화살표>

현재 읽기 포인터의 이전 객체를 읽은 후에 그 객체의 중간 위치로 읽기 포인터를 위치시킵니다. 이 기능은 현재 활성 윈도우 내의 객체만을 대상으로 동작합니다.

2. 현재 객체 읽기 <CTRL-Insert-Numpad5>

읽기 포인터가 위치한 현재 객체를 읽습니다.

3. 다음 객체로 <CTRL-Insert-Numpad6> <CTRL-Insert-오른쪽 화살표>

현재 읽기 포인터의 다음 객체를 읽은 후에 해당 객체의 중간 위치로 읽기 포인터를 위치시킵니다. 이 기능은 현재 활성 윈도우 내의 객체만을 대상으로 동작합니다.


Ⅷ-1-7. 포인터 기준 읽기


처음부터 포인터까지 읽기 <Alt-Numpad8>
왼쪽 읽기 <Alt-Numpad4>
오른쪽 읽기 <Alt-Numpad6>
포인터부터 끝까지 읽기 <Alt-Numpad2>

1. 처음부터 포인터까지 읽기 <Alt-Numpad8>

현재 지정된 윈도우 범위의 처음부터 읽기 포인터 위치의 이전까지 읽어줍니다.

2. 왼쪽 읽기 <Alt-Numpad4>

현재 읽기 포인터가 위치한 줄의 처음부터 읽기 포인터 위치의 이전까지 읽습니다.

3. 오른쪽 읽기 <Alt-Numpad6>

현재 읽기 포인터가 위치한 곳부터 줄의 끝까지 읽습니다.

4. 포인터부터 끝까지 읽기 <Alt-Numpad2>

읽기 포인터가 위치한 곳부터 현재 지정된 윈도우 범위의 끝까지 읽습니다.
