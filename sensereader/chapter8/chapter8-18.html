Ⅷ-18. 숫자 입력 관련 기능


  • Ⅷ-18-1. 왼쪽 숫자 입력 <Insert-Numpad/(슬래시)>
  • Ⅷ-18-2. 왼쪽 숫자 기억 <Ctrl-Numpad/(슬래시)>
  • Ⅷ-18-3. 기억된 숫자 입력 <Insert-Numpad*(스타)>
  • Ⅷ-18-4. 기억된 숫자 삭제 <Ctrl-Numpad*(스타)>

Ⅷ-18-1. 왼쪽 숫자 입력 <Insert-Numpad/(슬래시)>


읽기 포인터가 위치한 줄의 처음 숫자를 입력합니다.

통신상에서 게시물을 읽을 때 게시물 번호를 직접 타이핑하지 않고 이 기능을 이용하여 쉽게 게시물 번호를 입력할 수 있습니다.

만약 읽기포인터 현재 줄에 숫자가 발견되지 않으면 "숫자가 아닙니다"라는 음성 메세지를 출력합니다.


Ⅷ-18-2. 왼쪽 숫자 기억 <Ctrl-Numpad/(슬래시)>


읽기 포인터가 위치한 줄의 처음 숫자를 기억합니다.

통신 등에서 여러 개의 자료를 다운받을 때 자료들의 번호를 기억시켜 두는 기능입니다. 이 기능으로 기억한 숫자를 일괄적으로 입력하려면 '기억된 숫자 입력' 기능을 사용합니다.


Ⅷ-18-3. 기억된 숫자 입력 <Insert-Numpad*(스타)>


'왼쪽 숫자 기억' 기능에서 기억해 놓은 숫자들을 입력해 줍니다.


Ⅷ-18-4. 기억된 숫자 삭제 <Ctrl-Numpad*(스타)>


'왼쪽 숫자 기억' 기능으로 기억된 숫자들을 모두 삭제합니다.

자료들을 한 번 일괄적으로 다운받고 다시 다른 자료들을 '왼쪽 숫자 기억' 기능을 사용해서 다운받고자 한다면 이 기능을 이용해 먼저 기억해 두었던 숫자들을 삭제해 주어야 합니다.
