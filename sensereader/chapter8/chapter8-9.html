Ⅷ-9-1. 음성 출력 설정


음성출력설정 <없음>
음성 종류 선택 <Insert-→(오른쪽 화살표)>
음성 속성 선택 <Insert-←(왼쪽 화살표)>
음성 수치 증가 <Insert-↑(위쪽 화살표)>
음성 수치 감소 <Insert-↓(아래 화살표)>

1. 음성출력설정 <없음>

센스리더의 음성 종류에 따라 속도, 높이, 크기를 조절하는 대화상자를 호출합니다.
센스리더 [음성설정] 메뉴의 [음성출력]메뉴와 동일한 기능입니다.

2. 음성 종류 선택 <Insert-오른쪽 화살표>

음성 종류는 '마우스 | 키보드 | 캐럿 | 메시지 | 전체' 등으로 구성됩니다. '전체'는 음성 종류 전부를 포괄합니다. 음성 종류를 '전체'에 맞추어 놓고 음성 속도를 변경했다면 키보드, 마우스, 캐럿, 메시지 등의 모든 음성 속도가 변경됩니다.

3. 음성 속성 선택 <Insert-왼쪽 화살표>

음성 속성은 '높이 | 크기 | 속도' 등으로 구성됩니다. 현재 설정된 음성 종류에 해당되는 높이, 속도, 크기입니다.

4. 음성 수치 증가 <Insert-위쪽 화살표>

현재 선택된 음성 종류의 속성에 대한 값을 증가시킵니다. 만일 음성 종류는 '전체'이고 속성이 '속도 6'이라면 '전체 음성속도 7'로 변경됩니다. 전체란 마우스, 키보드, 캐럿, 메시지 등의 모든 음성 종류를 포괄하는 것입니다.

5. 음성 수치 감소 <Insert-아래 화살표>

현재 선택된 음성 종류의 속성에 대한 값을 감소시킵니다.


Ⅷ-9-2. 반복 문자 읽기 설정 <없음>


반복된 문자를 어떻게 읽을 것인가 설정합니다.

보다 자세한 내용은 '메뉴 구성 > 음성 설정 > 반복 문자 읽기 설정 항목을 참고하세요.


Ⅷ-9-3. 반복 문자 처리 토글 <없음>


반복문자 처리를 할 것인지 여부를 선택합니다.


Ⅷ-9-4. 반복 문자 음성 알림 토글 <Alt-Shift-A>


반복 문자 처리가 선택되었을 경우 음성으로 출력할지 여부를 선택합니다.


Ⅷ-9-5. 대문자 읽기 방법 설정 <없음>


대문자 읽기 방법은 '음 높이'와 'CAP 붙임'의 두 가지 옵션이 있습니다.

음 높이 설정의 경우 현재 높이가 최고 높이가 아니면 최고 높이로 읽으며, 현재 높이가 최고 높이이면 최저 높이로 읽습니다.

'CAP 붙임'은 대문자 앞에 'CAP' 문자를 붙여서 음성출력합니다.


Ⅷ-9-6. 영어 읽기 토글 <Alt-Shift-E>


영어를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-7. 축약 영단어 모아읽기


I'M, don't와 같은 축약 영단어의 읽기 방식을 지정합니다.
이 옵션이 선택되어 있는 경우, 구두점 읽기 옵션과 상관없이 축약 형태의 영단어를 만나면 어퍼스트로피 기호를 빼고 축약 형태로 음성을 출력합니다.(예: 아임, 돈트 등)
일반적인 영문 문서를 읽을 때는 이 옵션을 선택하는 것이 좋으며, 교정이나 세밀한 편집 작업을 하는 경우는 이 옵션을 해제하고 사용합니다.


Ⅷ-9-8. 숫자 읽기 토글 <Alt-Shift-V>


숫자를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-9. 숫자 모아읽기


구두점 읽기 방식과 상관없이 숫자를 모아서 읽기 원하는 경우 사용하는 옵션입니다.
이 옵션이 선택된 경우, 구두점 읽기 옵션이 선택되어 있더라도 숫자를 모아서 읽어줍니다. 반대로 이 옵션이 해제된 상태에서 구두점 읽기 옵션이 선택되어 있다면 기존 방식처럼 콤마 기호를 중심으로 세 자리씩 숫자를 끊어서 읽어줍니다.


Ⅷ-9-10. 구두점 읽기 토글 <Alt-Shift-Z>


구두점 읽기 목록의 설정에 따라 구두점 읽기 방법을 선택합니다.

기본적으로 센스리더는 '모두 읽기'와 '읽지 않기'의 두 가지 목록을 가지고 있습니다.

사용자가 필요에 따라 추가한 구두점 읽기 목록이 있다면 기능키를 연속해서 누를 때마다 하나의 구두점 읽기 목록이 선택됩니다.


Ⅷ-9-11. 한글 읽기 토글 <없음>


한글을 읽을 것인지 여부를 선택합니다.


Ⅷ-9-12. 한자 읽기 토글 <Alt-Shift-X>


한자를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-13. 한자 풀어읽기 선택


한자를 읽을 때 음만 읽을 것인지 음과 훈을 같이 읽을 것인지 선택합니다.

사용자가 선택할 수 있는 옵션은 다음 세 가지입니다.

- 해제: 한자의 음만 출력합니다. 정확한 한자의 훈을 확인하고자 한다면 풀어읽기 기능을 실행합니다.

- 글자 단위 : 라인 단위, 단어 단위 이동시에는 한자의 음만 출력하며, 글자 단위 이동시 한자를 만난 경우 한자의 음과 훈을 모두 출력합니다.

- 모두 : 모든 한자의 음과 훈을 풀어서 읽어줍니다.


Ⅷ-9-14. 공백 문자 읽기 토글 <없음>


'Space' 문자를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-15. 탭 문자 읽기 토글 <없음>

Ⅷ-9-16. 리턴 문자 읽기 토글 <없음>


리턴 문자를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-17. 이스케이프 문자 읽기 토글 <없음>


'Escape' 문자를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-18. 백스페이스 문자 읽기 토글 <없음>


'백스페이스' 문자를 읽을 것인지 아닌지를 선택합니다.


Ⅷ-9-19. 선 문자 읽기 토글 <Alt-Shift-C>


한자, 한글, 영문, 특수문자 등을 제외한 나머지 기호들을 ‘선’으로 읽을 것인지 여부를 선택합니다.


Ⅷ-9-20. 억양 넣음 토글 <없음>


문장 부호에 따라 억양을 사용할지 여부를 선택합니다.

억양 넣음이 선택되면 물음표가 붙은 문장은 올려 읽고, 마침표가 붙은 문장은 내려 읽습니다. 이 경우 대부분의 구두점은 읽지 않습니다.


Ⅷ-9-21. 사용자 단어 사전 사용 토글 <없음>

Ⅷ-9-22. 문자 사전 사용 토글 <없음>

Ⅷ-9-23. 특수 문자 사전 사용 토글 <Alt-Shift-'(어퍼스트로피)>


사용자가 지정한 특수 문자를 사용할 것인지 여부를 선택합니다.


Ⅷ-9-24. 특수문자 풀어읽기 선택


특수문자를 읽을 때 특수문자 사전 설정에서 지정한 내용만 읽을 것인지 풀어서 읽을 것인지 선택합니다.

예를 들어, '￦' 기호는 특수문자 사전 설정에 보면 내용 항목에는 '원화'로 등록되어 있고, 풀어읽기 항목에는 '전각 원화표'라고 등록되어 있습니다.

다음 세 가지 옵션 중 한 가지 출력 방식을 선택할 수 있습니다.

- 해제 : 특수문자를 만난 경우, 특수문자 사전에서 내용 항목에 지정된 값만을 출력합니다.

- 글자 단위 : 라인이나 단어 이동시에는 내용만을 출력하며, 글자 단위 이동시 풀어읽기 항목의 값을 읽어줍니다.

- 모두 : 특수문자를 만난 경우 풀어읽기 항목의 지정된 값을 출력합니다.


Ⅷ-9-25. 음성 세부 설정 <없음>


음성 설정의 세부 항목들을 설정합니다.

보다 자세한 내용은 '메뉴 구성 > 음성 설정 > 음성 세부 설정 항목을 참고하십시요.


Ⅷ-9-26. 보조 음성 엔진 모드 변경 <Insert-Ctrl-E>


보조 음성 엔진의 동작을 캐럿 자동모드/전체 자동 모드/수동 모드/해제 중에서 선택합니다.

- 자동 모드: 출력 언어를 자동 판단하여 해당 엔진으로 출력합니다. 자동 모드에서 사용할 엔진은 <보조음성엔진 선택> 대화상자를 이용하여
자동모드 사용 엔진으로 선택해 두어야 합니다. 예를 들어, <보조음성엔진 선택> 대화상자에서 영어엔진을 자동모드 사용 엔진으로 설정하였다면
한/영 혼용 문장에서 한글은 기본 음성엔진이 동작하고 영어는 영어 보조 음성엔진이 동작하게 됩니다. 해당 엔진이 없으면 기본엔진으로 출력됩니다.

* 캐럿 자동 모드에서는 가상커서의 내용과 편집기에서의 캐럿 이동에 대한 내용에 대해서 보조 음성 엔진을 적용하고, 전체 자동 모드는 메뉴 및 일반 메시지 내용 모두에 보조 음성 엔진을 적용합니다.

- 수동 모드: 출력엔진을 자동 변경하지 않고 <보조음성엔진 선택> 대화상자에서 지정된 엔진으로 출력합니다. 단, 인터넷에서 lang 속성이
지정되어 있는 경우는 lan 속성으로 지정된 언어의 엔진으로 출력됩니다.

- 해제: 보조음성엔진을 사용하지 않습니다.

* 보조음성엔진의 속도, 높이, 크기 등의 설정을 변경하려면 <보조음성엔진 선택> 대화상자를 호출하여 값을 설정합니다.


Ⅷ-9-27. 음성 사용 토글 <Alt-Shift-백스페이스>


센스리더의 음성출력만을 중지할 것인지 선택합니다.

'프로그램 사용 토글' 기능은 센스리더의 프로그램 기능을 정지하는 것에 비해 이 기능은 음성출력만을 정지한다는 것이 다릅니다.

윈앰프 방송처럼 센스리더의 음성출력이 문제가 되는 경우 음성출력을 중지시키고 '한소네' 같은 점자 디스플레이 장치로 센스리더의 메세지를 확인하며 컴퓨터를 사용할 수 있습니다.


Ⅷ-9-28. 음성엔진 선택 <없음>


이 기능은 센스리더의 음성엔진을 변경하는 대화상자를 호출하는 기능입니다.
센스리더 <일반> 메뉴의 <음성엔진변경> 메뉴로 진입해서 설정하는 것과 같은 기능을 합니다.


Ⅷ-9-29. 보조음성엔진 선택<없음>


이 기능은 센스리더의 보조음성엔진을 변경하고 속도/높이/크기를 조절하는 대화상자를 호출하는 기능입니다.
센스리더 <일반> 메뉴의 <보조음성엔진설정> 메뉴로 진입해서 설정하는 것과 같은 기능을 합니다.
