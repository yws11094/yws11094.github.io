Ⅷ-10. 환경 기능


  • Ⅷ-10-1. 환경 열기 <Ctrl-Alt-Shift-O> <Insert-O>
  • Ⅷ-10-2. 현재 환경 저장 <Ctrl-Alt-Shift-S>
  • Ⅷ-10-3. 환경 다른 이름으로 저장 <Ctrl-Alt-Shift-A>
  • Ⅷ-10-4. 현재 환경 재설정 <Insert-E>
  • Ⅷ-10-5. 기본 환경 열기 <Ctrl-Alt-Shift-B>

Ⅷ-10-1. 환경 열기 <Ctrl-Alt-Shift-O> <Insert-O>


센스리더의 환경을 열어줍니다.

보다 자세한 내용은 '6장 메뉴 구성 > 일반 > 열기' 항목을
참조하십시요.


Ⅷ-10-2. 현재 환경 저장 <Ctrl-Alt-Shift-S>


현재 사용중인 환경을 저장합니다.

좀 더 자세한 내용은 '메뉴 구성 > 일반 > 저장' 항목을
참고하십시요.


Ⅷ-10-3. 환경 다른 이름으로 저장 <Ctrl-Alt-Shift-A>


현재 열려져 있는 환경을 다른 이름으로 저장합니다.

좀 더 자세한 내용은 '메뉴 구성 > 일반 > 다른 이름으로 저장' 항목을 참고하십시요.


Ⅷ-10-4. 현재 환경 재설정 <Insert-E>


현재 열려 있는 센스리더의 환경을 다시 로딩합니다.

보다 자세한 내용은 '메뉴 구성 > 일반 > 환경 재설정' 항목을 참고하십시요.


Ⅷ-10-5. 기본 환경 열기 <Ctrl-Alt-Shift-B>


센스리더의 기본 환경(xvsrd)을 여는 기능입니다. 열기 대화상자를 통하지 않고 기능키만으로 기본 환경을 열어줍니다.
