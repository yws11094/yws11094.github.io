Ⅷ-8-1. 포커스 자동 추적 토글 <없음>


포커스 자동 추적이란 포커스된 객체에 읽기 포인터를 자동으로 위치시키는 기능입니다.

이 기능을 사용하면 읽기 포인터를 사용하여 화면을 읽을 때 읽기 포인터가 포커스된 객체로 자동으로 이동됨으로 포커스 객체부터 화면을 읽어야 하는 경우 편리합니다.


Ⅷ-8-2. 마우스 이동 자동읽기 토글 <없음>


실제 컴퓨터에 장착된 마우스가 이동되었을 경우 해당 위치의 내용을 읽어줍니다.

화면이 전혀 보이지 않는 사용자의 경우 사용하지 않는 것이 좋으며, 마우스를 사용할 수 있는 약시 사용자의 경우에는 사용하는 것이 좋습니다.


Ⅷ-8-3. DEL키 읽기 방법 설정 <없음>


DEL키 읽기 방법에는 '지운 글자 읽기'와 '지운 글자 다음 글자 읽기'가
있습니다.

지운 글자 읽기는 현재 캐럿이 있는 위치의 글자를 읽어주고, 지운 글자 다음 글자 읽기는 현재 글자가 지워진 후 현재 캐럿 위치에 오는 글자를 읽습니다.


Ⅷ-8-4. 숏컷 읽기 토글 <없음>


쇼컷을 읽을 것인지 아닌지를 선택합니다.


Ⅷ-8-5. 악셀러레이터키 읽기 토글 <없음>


악셀러레이터키를 읽을 것인지 아닌지 선택합니다.


Ⅷ-8-6. 컨트롤 종류 읽기 설정 <없음>


컨트롤들의 종류를 읽을 것인지 여부를 선택합니다.


Ⅷ-8-7. 툴팁 읽기 토글 <없음>


툴팁이 발생했을 때 툴팁을 읽을 것인지 아닌지를 선택합니다.


Ⅷ-8-8. 대화상자 자동 읽기 토글 <없음>


대화상자가 출력되었을 때 대화상자의 내용을 자동으로 읽어줄 것인지 아닌지를 선택합니다.


Ⅷ-8-9. 키 레이블 읽기 토글 <없음>


키 레이블 설정 기능에서 지정된 키 레이블을 읽을 것인지 아닌지를 선택합니다.


Ⅷ-8-10. 목록항목 레이블된 그래픽 읽기 토글 <없음>


목록상자에서 목록항목 이동시 목록의 그래픽 레이블을 읽을 것인지 여부를 선택합니다.

목록 항목들 중에는 목록 좌측에 그래픽을 표시하여 목록의 상태를 알려주는 경우가 있습니다. 이 그래픽에 레이블이 되어 있을 경우 목록항목을 읽을 때 레이블과 목록을 같이 읽을 것인지를 선택하는 것입니다.

이 기능은 '시스템 구성 유틸리티(msconfig)'의 시작 프로그램 목록들의 선택 여부를 표시하는 데 사용되고 있습니다.


Ⅷ-8-11. 빈줄 메세지 알림 토글 <없음>


캐럿 기능 사용시 빈줄에 대해 '빈줄' 메세지를 출력할 것인지 여부를 선택합니다.


Ⅷ-8-12. 캐럿 기능시 포인터 이동 토글 <없음>


캐럿 기능시 읽기 포인터를 캐럿 위치로 옮길 것인지 여부를 선택합니다.

캐럿 기능이란 캐럿 이전 글자 읽기, 캐럿 다음 단어 읽기, 캐럿 현재 줄 읽기처럼 캐럿을 움직여 문서를 읽는 동작을 말합니다.


Ⅷ-8-13. 시스템 하일라이트 사용 토글 <없음>


시스템이 지정한 기본 하일라이트를 윈도우 범위 읽기 방법 중 '하일라이트 읽기'로 설정했을 경우 사용할 것인지 여부를 선택합니다.


Ⅷ-8-14. 가상 커서 자동 사용 토글 <없음>


가상커서를 사용해야 하는 인터넷이나 웹 문서, PDF 문서 읽기 등에서 가상커서 기능을 자동으로 사용할지 여부를 선택합니다.


Ⅷ-8-15. 기능키 사용 토글 <없음>


기능키를 사용할 것인지 여부를 선택합니다.

여기서 기능키란 [환경설정] -> [기능키 설정]에서 목록에 표시된 기능키들을 의미하며, 가상커서의 기능키들은 해당되지 않습니다.


Ⅷ-8-16. 프로그램 사용 토글 <Alt-Shift-\(백슬래시)>


센스리더를 메모리에서 제거하지 않고 '프로그램 사용 토글' 기능을 제외한 모든 기능을 중지시킵니다.

다시 기능키를 누르면 센스리더가 정상적으로 동작합니다.


Ⅷ-8-17. 항목 개수 읽기 토글 <없음>


목록상자에서 현재 / 전체의 항목수를 읽을 것인지 여부를 선택합니다.


Ⅷ-8-18. 자동 시간 알림 토글 <없음>


자동으로 시각을 알려줄지 여부를 선택합니다.


Ⅷ-8-19. 자동 드래그 토글 <Alt-Shift-NumpadDel>


픽셀 이동시에 자동으로 드래그 기능을 사용할지 여부를
선택하는 기능입니다.

이 기능이 선택되어 있으면 픽셀 단위 이동(왼쪽, 오른쪽,
위, 아래) 기능을 사용할 때 이동된 거리만큼 드래그 하게
됩니다.


Ⅷ-8-20. 비디오 디바이스모드 토글 <Ctrl-Alt-Shift-V>


비디오 디바이스 모드를 사용할지 여부를 선택합니다.

디바이스 모드를 사용하면 센스리더가 화면상의 정보를 보다 정확하게 처리하며, 인터넷에서의 화면
깜박임 현상도 방지할 수 있습니다.

* 경우에 따라서는 이 모드를 사용하면 충돌이 발생하는 작업이 있을 수 있습니다. 이 경우 작업을 시작하기 전에 이 모드를 해제하고, 작업이 끝난 후 다시 모드를 선택하면 됩니다.


Ⅷ-8-21. 프로그레스바 자동 읽기 <없음>


인터넷에서 페이지가 열릴 때나 이야기멀티의 다운로드 대화상자, 프로그램 설치 대화상자 등에서 표시되는 진행 상황(프로그레스)을 자동으로 읽을 것인가를 선택합니다.

이 기능을 사용할 때는 프로그램에 따라 진행상황이 잘못 표시되는 경우도 있을 수 있습니다.
