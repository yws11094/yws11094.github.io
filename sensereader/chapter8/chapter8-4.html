Ⅷ-4-1. 캐럿 글자 단위 읽기


캐럿 이전 글자 읽기 <Ctrl-Shift-M>
캐럿 현재 글자 읽기 <Ctrl-Shift-,(쉼표)>
캐럿 다음 글자 읽기 <Ctrl-Shift-.(마침표)>

1. 캐럿 이전 글자 읽기 <Ctrl-Shift-M>

현재 캐럿 위치의 이전 글자를 읽습니다. 캐럿이 존재하지 않는 포커스 객체의 경우 객체의 내용을 한 글자 이전으로 이동하여 읽어주고, 엑셀에서는 셀의 내용을 한 글자 이전으로 이동하며 읽어줍니다.

2. 캐럿 현재 글자 읽기 <Ctrl-Shift-,(쉼표)>

현재 캐럿 위치의 글자를 읽습니다. 캐럿이 존재하지 않는 객체의 경우 포커스 객체의 내용을 중심으로 현재 글자를 읽어주고, 엑셀에서는 현재 셀의 내용을 기준으로 현재 글자를 읽어주게 됩니다. 이 기능은 반복해서 기능키를 누르면 현재 글자를 풀어서 읽어주며, 가상커서에서도 사용 가능합니다.

3. 캐럿 다음 글자 읽기 <Ctrl-Shift-.(마침표)>

현재 캐럿 위치의 다음 글자를 읽어줍니다. 캐럿이 존재하지 않는다면 포커스 객체의 내용을 한 글자 다음으로 이동하며 읽어주고, 엑셀에서는 현재 셀의 내용을 한 글자 다음으로 이동하며 음성출력합니다.


Ⅷ-4-2. 캐럿 단어 단위 읽기


캐럿 이전 단어 읽기 <Ctrl-Shift-J>
캐럿 현재 단어 읽기 <Ctrl-Shift-K>
캐럿 다음 단어 읽기 <Ctrl-Shift-L>

1. 캐럿 이전 단어 읽기 <Ctrl-Shift-J>

현재 캐럿 위치의 이전 단어를 읽습니다. 캐럿이 없는 포커스 객체의 경우 포커스 내용을 한 단어 이전으로 이동하며 읽어주고, 엑셀에서는 한 셀씩 좌측으로 이동하며 음성출력합니다.

2. 캐럿 현재 단어 읽기 <Ctrl-Shift-K>

캐럿 위치의 현재 단어를 읽습니다. 캐럿이 존재하지 않는다면 포커스 객체 내용을 중심으로 현재 단어 읽기를 하고, 엑셀에서는 현제 셀을 읽습니다. 캐럿 현재 단어 읽기를 연속해서 실행하면 단어를 자세히 풀어서 읽어줍니다.

3. 캐럿 다음 단어 읽기 <Ctrl-Shift-L>

현재 캐럿 위치의 다음 단어를 읽습니다. 캐럿이 존재하지 않는다면 포커스 내용을 한 단어 다음으로 이동하며 읽고, 엑셀에서는 오른쪽 방향으로 한 셀씩 이동하며 읽습니다.


Ⅷ-4-3. 캐럿 줄 단위 읽기


캐럿 이전 줄 읽기 <Ctrl-Shift-U>
캐럿 현재 줄 읽기 <Ctrl-Shift-I>
캐럿 다음 줄 읽기 <Ctrl-Shift-O>

1. 캐럿 이전 줄 읽기 <Ctrl-Shift-U>

현재 캐럿이 위치한 줄의 이전 줄을 읽습니다. 엑셀에서는 윗 방향으로 한 셀씩 이동하며 읽어줍니다.

2. 캐럿 현재 줄 읽기 <Ctrl-Shift-I>

현재 캐럿이 위치한 라인 전체를 읽습니다. 캐럿이 존재하지 않는다면 현재 포커스 객체를 읽어주고, 가상커서에서는 현재 가상커서가 위치한 줄을 읽습니다. 엑셀의 경우 현재 포커스된 셀이 위치한 라인을 음성출력합니다.

3. 캐럿 다음 줄 읽기 <Ctrl-Shift-O>

현재 캐럿 위치의 다음 줄을 읽습니다. 엑셀에서는 아래 방향으로 한 셀씩 이동하며 음성출력을 합니다.


Ⅷ-4-4. 캐럿 기준 읽기


처음부터 캐럿까지 읽기 <Ctrl-Shift-위쪽 화살표>
캐럿 왼쪽 읽기 <Ctrl-Shift-왼쪽 화살표>
캐럿 오른쪽 읽기 <Ctrl-Shift-오른쪽 화살표>
캐럿부터 끝까지 읽기 <Ctrl-Shift-아래쪽 화살표>

1. 처음부터 캐럿까지 읽기 <Ctrl-Shift-위쪽 화살표>

화면에 나타난 편집모드 내용의 처음부터 캐럿 위치의 이전 글자까지 음성으로 출력합니다.

2. 캐럿 왼쪽 읽기 <Ctrl-Shift-왼쪽 화살표>

현재 캐럿이 위치한 줄의 처음부터 캐럿 이전 글자까지 읽습니다.

3. 캐럿 오른쪽 읽기 <Ctrl-Shift-오른쪽 화살표>

현재 캐럿이 위치한 글자부터 줄의 끝까지 읽습니다.

4. 캐럿부터 끝까지 읽기 <Ctrl-Shift-아래쪽 화살표>

현재 캐럿 위치의 글자부터 화면에 나타난 편집모드 내용의 끝까지 읽습니다.
