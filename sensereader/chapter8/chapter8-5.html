Ⅷ-5-1. 키보드를 활용한 객체 탐색 기능


해당 기능을 활용하여 생성된 윈도우 객체를 자유롭게 이동하며 내용을 확인하고 기능을 실행할 수 있습니다.
예를 들어 기존에 키패드키를 활용한 리뷰기능만으로 접근하기 어려웠던 내용들을 객체 탐색 기능을 활용하여 확인할 수 있습니다.

1. 이전 객체로(Ctrl-Insert-Numpad4, Ctrl-Insert-왼쪽 화살표)

탐색 포인터를 이전 객체로 이동합니다.

2. 현재 객체읽기(Ctrl-Insert-Numpad5)

탐색 포인터가 위치한 객체의 정보를 읽어 줍니다.

3. 다음 객체로(Ctrl-Insert-Numpad6, Ctrl-Insert-오른쪽 화살표)

탐색 포인터를 이전 객체로 이동합니다.

4. 상위 객체로(Ctrl-Insert-Numpad8, Ctrl-Insert-위쪽 화살표)

현재의 객체를 포함하고 있는 상위 레벨의 객체로 이동합니다.

5. 첫 하위 객체로(Ctrl-Insert-Numpad2, Ctrl-Insert-아래쪽 화살표)

현재의 객체가 포함하고 있는 첫 하위 레벨의 객체로 이동합니다.

6. 객체 활성(Ctrl-Insert-NumpadEnter, Ctrl-Insert-Enter)

탐색 포인터가 위치한 현재의 객체를 실행합니다.

7. 객체 포커스(Ctrl-Insert-Numpad+(더하기), Ctrl-Insert-Space Bar)

현재의 객체가 키보드 입력을 받을 수 있도록 포커스를 지정합니다.

8. 하위 객체 자동 탐색 토글(Ctrl-Insert–Numpad-(대시))

이전, 다음 객체 이동시에 전체 객체를 탐색할 것인지, 상위 객체에 포함된 같은 레벨의 객체만 탐색할 것인지를 토글합니다.

Ⅷ-5-2. 화면터치를 활용한 객체 탐색 기능


터치모니터 사용이 가능한 윈도우 8, 8.1, 10환경에서 터치 제스처를 활용하여 객체 탐색 기능을 사용할 수 있습니다.
사용 가능한 제스쳐는 다음과 같습니다.

{{1. 이전 객체로(한 손가락 왼쪽 쓸기)}}
탐색 포인터를 이전 객체로 이동합니다.

{{2. 현재 좌표 객체 읽기(한 손가락 탭 또는 마우스나 손가락 대고 이동)}}
손가락으로 탐색중인 위치의 객체 정보를 읽습니다.

{{3. 다음 객체로(한 손가락 오른쪽 쓸기)}}
탐색 포인터를 다음 객체로 이동합니다.

{{4. 상위 객체로(한 손가락 위로 쓸기)}}
현재의 객체를 포함하고 있는 상위 레벨의 객체로 이동합니다.

{{5. 첫 하위 객체로(한 손가락 아래로 쓸기)}}
현재의 객체가 포함하고 있는 첫 하위 레벨의 객체로 이동합니다.

{{6. 객체 활성(한 손가락 두 번 탭 또는 한 손가락 대고 다른 한 손가락 탭)}}
탐색 포인터가 위치한 현재의 객체를 실행합니다.

{{7. 객체 포커스(세 손가락 두 번 탭)}}
현재의 객체가 키보드 입력을 받을 수 있도록 포커스를 지정합니다.

{{8. 하위 객체 자동 탐색 토글(네 손가락 두 번 탭)}}
이전, 다음 객체 이동시에 전체 객체를 탐색할 것인지, 상위 객체에 포함된 같은 레벨의 객체만 탐색할 것인지를 토글합니다.

{{9. 현재 활성창 닫기(네 손가락 아래로 쓸기 또는 한 손가락 대고 세 손가락 두 번 탭)}}
현재 열려 있는 창을 닫습니다.(키보드의 Alt-F4 기능과 같습니다.)

{{10. 음성 중지(두 손가락 한번 탭)}}
출력중인 음성을 중지합니다.

{{11. 팝업메뉴 호출(한 손가락 세 번 탭 또는 한 손가락 대고 다른 한 손가락 두 번 탭 = 팝업 메뉴 호출)}}
팝업 메뉴를 호출 합니다.
※ 12. 기타 터치 재스처에 대한 자세한 내용은 터치 재스처 이용 방법에서 확인하시기 바랍니다.


Ⅷ-5-3. 객체 탐색 모드 토글


리뷰 모드에서 사용되는 기능키를 이용하여 윈도우 객체를 탐색할 수 있도록 토글합니다.

센스리더의 마우스 동작 기능을 이용하여 객체를 클릭할 수도 있으며, 모던 UI, Internet Explorer 9 이상, 파이어폭스, 크롬, 카카오톡 등의 리뷰 모드 기능이 지원 되지 않는 응용프로그램을 객체 탐색 모드로 접근할 수 있습니다.
또한 센스월드, 모던 UI, 익스플로러, 크롬, 파이어폭스를 실행시 객체탐색 모드는 기본으로 실행되며, 디스플레이 모드는 객체 탐색 모드가 비활성화된 상태로써 원래의 리뷰 기능이 동작하는 모드입니다.

Desktop 모드 (Shift- (Numpad-(대시), Shift-Alt-[(대괄호 열고))

Desktop 모드에서 디스플레이 모드와 객체 탐색 모드를 토글합니다.

laptop 모드 (Shift-Alt-(대시), Shift-Alt-[(대괄호 열고))

laptop 모드에서 디스플레이 모드와 객체 탐색 모드를 토글합니다.
