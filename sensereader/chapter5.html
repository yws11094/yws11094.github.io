Ⅴ-1. 기본 기능 둘러보기


센스리더를 정상적으로 설치했다면 이제 센스리더의 기본적인 동작원리와 사용법을 안내해 드리겠습니다.
이번 장에서 다루는 내용은 센스리더를 이용하는 가장 기본적인 방법이며, 센스리더의 다양한 세부 기능에 대해서는 이후의 챕터들을 참고하십시요.

윈도를 사용자 입장에서 간략히 설명하면, 라디오버튼, 콤보상자, 목록상자, 편집창 등의 컨트롤들이 모여 하나의 윈도(창)를 구성하고, 이러한 윈도들의 집합체로써 윈도의 구성을 이해할 수 있습니다.
윈도의 동작은 거의 대부분 마우스를 통해 이루어지나 키보드의 동작만으로도 대부분의 기능 조작이 가능합니다.
센스리더는 윈도(창)의 변화나 메뉴의 변화, 포커스의 이동 등의 정보를 음성출력 함으로써 시각장애인들이 윈도를 사용할 수 있도록 도와줍니다.
사용자는 센스리더가 출력하는 음성 메시지를 정확하게 이해하므로써 윈도의 정상적인 사용이 가능해집니다.


1. 윈도(창)의 변화에 따른 동작

윈도에서 어떤 응용프로그램을 실행하게 되면 대부분 응용프로그램들은 화면에 윈도라는 사각 영역을 만들고, 윈도 안에서 자신의 정보와 필요한 메시지 그리고 동작상태 등을 표시하게 됩니다.
센스리더는 어떤 응용프로그램이 실행되어 새로운 창이 열리거나 세부 기능을 실행하여 대화상자가 열리게 되면 윈도의 제목줄(타이틀바)과 더불어 윈도의 모든 내용을 자동으로 읽어주게 됩니다.

현재 포커스된 윈도를 활성윈도라 하며, 센스리더에서 활성윈도의 정보를 확인하는 기능은 다음과 같습니다.

1) 제목표시줄(타이틀바) 읽기
제목표시줄에는 현재 실행 중인 프로그램의 이름과 열려진 파일의 이름 같은 중요한 정보들이 표시됩니다. 사용자는 제목표시줄을 확인하므로써 현재 내가 어떤 프로그램에 위치하고 있는지와 작업대상이 되는 파일이 무엇인지 등의 정보를 확인할 수 있습니다.
제목표시줄을 확인하는 단축키는 <Ctrl-Shift-T> 키이며, 센스리더의 활성 윈도의 정보를 확인하는 기능들은 대부분 <Ctrl>과 <Shift> 키의 조합으로 구성됩니다.

** 단축키를 누르는 방법: <Ctrl-Shift-T>키를 사용자가 누를 때 세 개의 키를 동시에 누르려고 하다 보면 정확한 동작이 이루어지지 않을 수 있습니다.
<Ctrl>과 <Alt> 그리고 <Shift> 키들은 이들 키만을 누를 때는 아무런 동작을 하지 않습니다. 그러므로 <Ctrl-Shift-T>키를 누른다면 <Ctrl> 키와 <Shift> 키를 먼저 누르고 있는 상태에서 알파벳 <T> 키를 눌렀다가 세 개의 키에서 동시에 손을 떼면 좀 더 손쉽게 기능키를 활용할 수 있습니다.

2) 상태표시줄 읽기
상태표시줄은 대개 윈도의 최하단에 존재하며, 프로그램 명과 파일명 외에 추가적인 정보를 표시합니다.
예를 들어 텍스트 편집 프로그램들의 경우 현재 커서가 위치하고 있는 라인이나 칼럼, 파일의 크기 등의 정보를 표시합니다.
센스리더에서 상태표시줄을 확인하는 단축키는 <Ctrl-Shift-S>입니다.

3) 윈도 내용 읽기
현재 활성화된 윈도에 표시되고 있는 내용을 전체적으로 확인할 때 사용하는 기능입니다.
단축키는 <Ctrl-Shift-W>키입니다.


2. 객체단위의 포커스 이동

프로그램 윈도나 대화상자는 여러 개의 컨트롤이 창에 배치되어 있으며 대부분의 프로그램에서는 <Tab> 키나 <Shift-Tab> 키의 조작으로 각 컨트롤을 이동할 수 있습니다.
센스리더는 어떤 컨트롤에 포커스가 되면 해당 컨트롤에 설정된 값과 종류 등을 음성으로 출력합니다.
예를 들어, 파일 열기 대화상자 내의 '파일 형식' 콤보상자에 포커스된 경우 "파일 형식(T): 텍스트 문서(*.txt) 콤보상자 = Alt+t"과 같이 음성출력을 하게 됩니다.
현재 포커스된 객체를 다시 확인하려면 <Ctrl-Shift-F> 단축키를 사용합니다.


3. 메뉴의 변화

메뉴는 크게 시작메뉴와 풀다운메뉴, 팝업메뉴 등으로 나눌 수 있습니다.
메뉴가 새로 열리게 되면 센스리더는 메뉴의 활성 상태와 포커스된 첫 번째 메뉴 항목을 읽어주게 됩니다. 또한, 특정 메뉴가 하위 메뉴를 가지고 있는 경우 '풀다운 메뉴'로 음성출력되며, 메뉴 실행시 대화상자가 열리는 메뉴라면 '대화상자'라는 음성 메시지를 메뉴 이름 뒤에 붙여서 출력하게 됩니다.


Ⅴ-2. 읽기포인터를 이용한 활용


읽기포인터는 화면을 자세히 살펴보거나 어떤 응용프로그램이 키보드의 기본키만으로 조작이 힘든 경우에 사용합니다.
읽기포인터에서 사용되는 키는 키보드 우측에 자리잡고 있는 뉴메릭 키패드(숫자판)이며, Numlock 키가 off된 상태에서 조작해야 합니다.


1. 줄 단위 이동

뉴메릭 키패드의 위, 아래 방향키를 누르면 각각 이전 줄과 다음 줄을 확인할 수 있습니다. 키패드의 중앙에 위치한 5번은 현재 라인을 다시 읽어줍니다.

2. 글자 단위 이동

뉴메릭 키패드의 좌, 우 방향키를 누르면 각각 왼쪽과 오른쪽의 글자를 확인할 수 있습니다. <Ctrl>키나 키패드의 <Insert>키를 누른 상태에서 좌, 우 방향키를 누르면 단어단위로 이동됩니다.

3. 마우스 버튼 동작

센스리더에서는 마우스 버튼의 조작이 필요할 경우 뉴메릭 키패드의 '/'(왼쪽 버튼 클릭) 키와 '*'(오른쪽 버튼 클릭) 키를 사용합니다. 읽기포인터의 방향키를 이용하여 원하는 버튼에 위치한 후 '/' 키와 '*'키를 누르면 직접 마우스로 조작하는 것과 같은 역할을 합니다.


Ⅴ-3. 인터넷의 활용(가상커서)


인터넷이나 PDF 문서는 시각장애인들이 사용하기 힘든 구조로 이루어져 있습니다. 센스리더에서는 이들에 대한 접근성을 높이기 위해 가상커서라는 새로운 사용자 인터페이스를 제공합니다.
프로그램 이용 중 '페이지 열림'이라는 음성안내가 나오면 가상커서가 동작했음을 의미합니다. 가상커서는 일반적으로 문서편집기와 비슷한 형태로 동작하며, 폼컨트롤이나 기타 구성요소 제어를 위한 추가적인 기능을 가지고 있습니다.


1. 방향키의 활용

일반적인 문서편집기의 동작과 마찬가지로 방향키를 이용하여 라인단위, 단어단위, 글자단위의 이동이 가능합니다.

2. Enter 키의 활용

엔터키는 현재 위치하고 있는 곳이 링크나 버튼이면 해당 개체를 실행시키고, 멀티라인 편집창에서는 가상커서를 해제하여 문자 입력이 가능한 상태로 전환해 줍니다.

3. Tab 키의 활용

홈페이지의 링크나 편집창, 라디오버튼, 체크상자 등의 객체 단위로 이동합니다.

4. 편리한 이동키들

홈페이지 문서들은 텍스트 외에 각종 폼컨트롤들이 위치하고 있어 읽기를 원하는 부분으로 빠르게 이동할 수 있는 방법이 중요합니다. 센스리더에서는 다음과 같은 기능들을 제공합니다.

  1. 폼컨트롤 이동 : 홈페이지상에서 <N>키를 누르면 폼컨트롤 단위로 이동합니다.
  2. 테이블 이동 : <T> 키를 누르면 테이블(표) 단위로 이동하게 됩니다.
  3. 텍스트 단위 이동 : <F4> 키를 누르면 웹페이지에서 텍스트 단위로 이동합니다.
  4. 헤딩 단위 이동 : <H> 키를 누르면 웹페이지에서 지정된 헤딩 단위로 이동합니다.
  5. 외부 객체 단위 이동 : <E> 키를 누르면 웹페이지에서 플래시 객체 단위로 이동합니다.
  6. 편집창 단위 이동 : <F2> 키를 누르면 웹페이지에서 편집창 단위로 이동합니다.
  7. 가상커서 사용/사용안함 : <Ctrl-Shift-F11>/<Ctrl-Shift-F12> 키를 누르면 가상커서를 사용하거나 해제할 수 있습니다.

* 위에서 제시된 이동 단축키들은 <Shift>키와 함께 사용하면 역방향으로 이동할 수 있습니다. (가상커서 사용/사용안함 제외)


Ⅴ-4. 독서기의 활용


센스리더는 사용자가 좀 더 편리하게 독서를 할 수 있도록 독서 전용 프로그램인 '센스 독서기'를 별도의 프로그램으로 제공합니다. 독서기는 크게 탐색창과 독서창으로 구성되어 있습니다.
바탕화면에서 '센스 독서기' 아이콘을 찾아 실행하면 탐색창이 실행되고, 탐색창에서 읽기 원하는 문서나 미디어 파일에 위치한 후 <Enter> 키를 누르면 독서창이 열리면서 해당파일의 내용을 출력하게 됩니다.


1. 탐색창의 활용

독서기의 탐색창에서는 윈도 탐색기와 비슷하게 드라이브나 폴더를 이동할 수 있고, 복사나 잘라내기, 삭제 같은 파일관리 기능이 가능합니다. 탐색창에서 사용할 수 있는 기능은 다음과 같습니다.

(1) 드라이브 이동
<Shift> 키를 누른 상태에서 이동을 원하는 드라이브 문자를 입력하면 해당 드라이브로 이동됩니다.

(2) 폴더 이동
특정 폴더로 진입을 원할 경우, 방향키를 이용해 해당 폴더까지 이동한 후 <Enter>키를 누릅니다. 상위 폴더로 나가려면 <BackSpace>키를 누르면 됩니다.

(3) 파일 관리 기능
파일이나 폴더를 이동, 복사, 붙여넣기, 삭제하려면 윈도 탐색기와 마찬가지로 <Ctrl-X>, <Ctrl-C>, <Ctrl-V>, <Delete> 키를 각각 누르면 됩니다.


2. 독서창의 활용

탐색창에서 읽기 원하는 파일에 위치한 후 <Enter> 키를 누르면 독서창이 열리면서 문서 파일이나 미디어 파일을 재생해 줍니다.
센스 독서기에서 지원하는 파일형식은 MP3, AVI, OGG 등의 기본적인 미디어파일과 조합형, 완성형, 유니코드 텍스트 그리고 DOC, HWP 등의 문서파일입니다.

독서창에서 사용 가능한 기능은 다음과 같습니다.

(1) 재생과 정지
독서기에서 재생 가능한 파일은 독서창이 열리면서 바로 재생이 시작됩니다. 재생 도중 정지를 하고 싶다면 <Enter> 키를 누르면 되고, 정지 위치에서 다시 재생하려면 다시 <Enter> 키를 누릅니다. 일시 정지는 <Space> 키를 사용합니다.

(2) 책갈피 기능
문서 파일이나 미디어 파일 재생 도중 <S> 키를 누르면 현재 위치에 책갈피가 설정됩니다. 책갈피가 설정된 파일은 다음 번 재생시 책갈피 위치부터 자동으로 재생됩니다.

(3) 파일 내에서의 이동
문서파일이나 미디어파일 모두 위/아래/오른쪽/왼쪽 방향키를 이용해 원하는 위치로 빠르게 이동할 수 있습니다.


Ⅴ-5. 그 외의 기능들


다음은 센스리더 활용 중 사용 빈도가 비교적 높은 기능들입니다.


1. 구두점 읽기/읽지않기

구두점을 빼고 읽을 것인가 아니년 구두점을 넣고 읽을 것인가를 선택합니다.
단축키는 <Alt-Shift-Z>키입니다. 구두점은 키보드에서 직접 입력할 수 있는 마침표, 쉼표, 사칙연산, 괄호 문자 등의 기호들을 의미합니다.

2. 특수문자 읽기/읽지않기

특수문자를 읽을 것인가 읽지 않을 것인가를 선택합니다. 단축키는 <Alt-Shift-'(어포스트로피)>입니다. 특수문자는 키보드에서 직접 입력할 수 없는 로마 숫자, 원문자 등의 기호들을 의미합니다.

3. 시각 확인

<Alt-Shift-T> 키를 누르면 현재 시각을 확인할 수 있습니다.

4. 날짜 확인

<Alt-Shift-D> 키를 누르면 현재 날짜를 확인할 수 있습니다.
