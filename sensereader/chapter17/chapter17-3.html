ⅩⅦ-3-1. 엑셀, 한셀 환경설정 실제


이번 장에서는 엑셀, 한셀에서 자주 사용되는 환경설정 방법을 설명드립니다. 몇 개의 환경설정을 연습해 보시면 사용자가 원하는 환경을 쉽게 구성할 수 있습니다.

1. 빈 셀 알림 메세지 없애기

기본 설정에서 셀 포인터의 상하, 좌우 이동시, 빈 셀을 만나더라도 '빈 셀'이라는 음성출력을 원치 않을 때 세부 옵션을 변경해 줍니다.
세부 옵션을 변경하는 방법은 '엑셀, 한셀 설정' 대화상자에서 직접 변경하는 방법과 '설정 목록' 대화상자를 호출하여 변경하는 방법이 있습니다. '엑셀, 한셀 설정' 대화상자에서 세부 옵션을 변경할 경우 현재 선택된 설정 전체의 메세지나 키정의에 모두 적용되며, '설정 목록' 대화상자를 호출하여 변경할 경우 해당되는 메세지나 키정의 내에서만 세부 옵션이 변경됩니다. 

방법 1 : 설정 대화상자에서 세부 옵션 조절

① 엑셀, 한셀이 실행된 상태에서 <Ctrl-Shift-F9> 키를 눌러 '엑셀, 한셀 설정' 대화상자를 엽니다.
② <Tab> 키를 눌러 기능 목록상자로 이동한 다음 '셀 포인터 내용 읽기' 항목에 위치합니다.
③ 다시 <Tab> 키를 눌러 세부 옵션 목록까지 이동한 뒤, '빈 셀 알림' 항목을 <Space> 키를 눌러 해제합니다.
④ '확인' 버튼을 눌러 설정을 적용합니다.

방법 2 : 설정 목록 대화상자에서 세부 옵션 조절

① 엑셀, 한셀이 실행된 상태에서 <Ctrl-Shift-F9> 키를 눌러 '엑셀, 한셀 설정' 대화상자를 엽니다.
② 목록상자에서 '자동 반응 설정'에 위치한 다음, <POPUP> 키를 눌러 팝업 메뉴를 엽니다.
③ 팝업 메뉴 중 [설정 목록] 메뉴를 실행합니다. '자동 반응 설정 목록' 대화상자가 출력됩니다.
④ 이름 목록상자에서 '기본'에 위치한 다음, <TAB> 키를 눌러 이벤트 목록상자로 이동합니다.
⑤ 이벤트 목록상자에서는 셀 포인터 이동을 선택해야 하는데, '셀 포인터 상/하 이동'과 '셀 포인터 좌/우 이동'으로 분리된 것에 주의합니다. 우선 '셀 포인터 상/하 이동'에 포커스를 맞춥니다.
⑥ 연결 기능 목록상자로 이동한 다음, '셀 포인터 내용 읽기'를 선택합니다.
⑦ 다시 세부설정 목록상자로 이동하여 '빈 셀 알림' 항목에 위치합니다.
⑧ <Space> 키를 눌러 '빈 셀 알림'의 선택을 해제합니다.
⑨ 이벤트 목록상자로 다시 이동하여 '셀 포인터 좌/우 이동' 항목에 포커스를 합니다.
⑩ '셀 포인터 상/하 이동' 설정과 마찬가지로 '셀 포인터 내용 읽기'를 선택하고,'빈 셀 알림'의 선택을 해제합니다.
⑪ '확인' 버튼을 계속해서 눌러 엑셀, 한셀의 메인화면으로 돌아갑니다.

2. 셀 제목 읽기 설정

워크시트에서 셀 포인터 이동시, 사용자가 등록한 기준 제목 셀의 내용을 읽으려면 다음의 순서에 따라 환경 설정을 해 줍니다.
① 엑셀, 한셀 문서를 열고, 제목 기준 셀로 등록할 임의의 셀로 이동합니다.
② <Ctrl-Shift-`(그레이브)> 키를 눌러 현재 위치한 셀을 제목 기준 셀로 등록합니다.
③ <Ctrl-Shift-F9> 키를 눌러 '엑셀, 한셀 설정' 대화상자를 엽니다.
④ 자동 반응 설정 목록에 위치한 다음 <Space> 키를 눌러 '기본(제목셀읽기)' 항목을 선택합니다.
⑤ '확인' 버튼을 눌러 대화상자를 닫습니다.
워크시트에서 셀 포인터를 움직이면 기준으로 등록한 제목 셀을 읽어줍니다.

<a name="chapter2"><h3>ⅩⅦ-3-2. 엑셀, 한셀 이벤트</h3>

자동 반응 설정에서 지원하는 엑셀, 한셀 이벤트는 다음과 같습니다.


1. 문서창 활성

엑셀, 한셀 문서창으로 활성이 변경된 경우 발생합니다.

2. 셀 포인터 상/하 이동

워크시트에서 셀 포인터가 수직 이동한 경우 발생합니다.

3. 셀 포인터 좌/우 이동

워크시트에서 셀 포인터가 수평 이동한 경우 발생합니다.

4. 선택 영역 변경

워크시트에서 선택영역이 추가, 해제될 때 발생합니다.(블럭 작업)

5. 활성 시트 변경

다른 시트로 활성 전환된 경우 발생합니다.


ⅩⅦ-3-3. 엑셀, 한셀 기능 리스트


센스리더는 EXCEL, 한셀에 대해 다음의 기능들을 지원합니다. 여기에 나열된 기능들은 자동 반응 설정이나 키 반응 설정에서 연결
기능으로 지정되어야 실제 동작을 하게 됩니다. 기능들 중에는 세부 동작을 지정하기 위해 세부 옵션이나 팝업 메뉴를 가지고 있는 기능들이 있습니다.

[] 자동 반응 설정 선택
설정 대화상자를 열지 않고 자동 반응 설정에 지정된 설정을 변경합니다.

[] 키 반응 설정 선택
설정 대화상자를 열지 않고 키 반응 설정에 지정된 설정을 변경합니다.

[] 현재 문서 이름 읽기
활성된 문서의 이름(파일이름)을 읽어 줍니다.

[] 현재 시트 이름 읽기
활성된 시트의 이름을 읽어줍니다.

[] 셀 포인터 주소 읽기
셀 포인터의 위치를 'A1'방식으로 알려줍니다.

[] 셀 포인터 내용 읽기
셀 포인터 위치의 셀 내용을 읽어줍니다.
-- 세부 옵션 --
1) 빈 셀 알림: 셀에 내용이 없을 경우 "빈셀"이라고 알려줄지 여부를 선택합니다.
2) 수식 알림: 수식이 있는 경우 "수식셀"이라 알려줄지 여부를 선택합니다.
3) 이름 알림: 이름이 등록되어 있는 경우 "이름셀"이라 알려줄지 여부를 선택합니다.
4) 메모 알림: 메모가 지정되어 있는 경우 "메모셀"이라고 알려줄지 여부를 선택합니다.
5) 링크 알림: 링크가 지정되어 있는 경우 "링크셀"이라고 알려줄지 여부를 선택합니다.

[] 셀 포인터 표시형식 읽기
셀 포인터의 표시형식(숫자, 통화, 날짜, 시간, 회계, 백분률 등)을 읽어줍니다.

[] 셀 포인터 맞춤 읽기
셀 포인터의 맞춤 형식을 읽어줍니다.
-- 세부 옵션 --
1) 가로 정렬 읽기: 가로 정렬 정보를 읽어줄지 여부를 선택합니다.
2) 세로 정렬 읽기: 세로 정렬 정보를 읽어줄지 여부를 선택합니다.
3) 들여쓰기 읽기: 들여쓰기 정보를 읽어줄지 여부를 선택합니다.
4) 텍스트 방향 읽기: 텍스트 회전 도수를 읽을지 여부를 선택합니다.
5) 텍스트 줄 바꿈 읽기: 텍스트 줄바꿈이 선택되어 있는 경우 이를 알려줄지 여부를 선택합니다.
6) 셀에 맞춤 읽기: 셀에 맞춤이 선택되어 있는 경우 이를 알려줄지 여부를 선택합니다.
7) 읽기 순서 읽기: 좌에서 우로, 우에서 좌로 등 텍스트 읽기 순서를 알려줄지 여부를 선택합니다.

[] 셀 포인터 글꼴 읽기
셀 포인터 위치 셀에 지정된 글꼴을 읽어줍니다.
-- 세부 옵션 --
1) 글꼴 이름 읽기: 글꼴의 이름(돋움, 돋움체, 바탕, 굴림, 굴림체 등)을 읽어줄지 여부를 선택합니다.
2) 글꼴 스타일 읽기: 글꼴 스타일(기울임꼴, 굵게, 굵은 기울임꼴) 등을 읽을지 여부를 선택합니다.
3) 글꼴 크기 읽기: 글꼴의 크기(포인트)를 읽을지 여부를 선택합니다.
4) 취소선 읽기: 취소선이 선택된 경우 이를 알려줄지 여부를 선택합니다.
5) 아래 첨자 읽기: 아래 첨자가 선택된 경우 이를 알려줄지 여부를 선택합니다.
6) 위 첨자 읽기: 위 첨자 읽기가 선택된 경우 이를 알려줄지 여부를 선택합니다.
7) 밑줄 읽기: 밑줄이 설정된 경우 밑줄 종류를 읽어줄지 여부를 선택합니다.
8) 글꼴 색 읽기: 글꼴 색을 읽어줄지 여부를 선택합니다.

[] 셀 포인터 태두리 읽기
셀 포인터 위치 셀의 태두리 정보를 읽어줍니다.
-- 세부 옵션 --
1) 태두리 색 읽기: 태두리에 지정된 색을 읽어줄지 여부를 선택합니다.

[] 셀 포인터 무늬 읽기
셀 포인터 위치 셀의 무늬 정보를 읽어줍니다.
-- 세부 옵션 --
1) 배경색 읽기: 배경색을 읽어줄지 여부를 선택합니다.

[] 셀 포인터 위치 읽기
셀 포인터 위치 셀의 위치(시트 좌상단에서 셀 좌상단까지의 거리)와 넓이, 높이 정보를 포인트 단위로 읽어줍니다.
-- 세부 옵션 --
1) 왼쪽 읽기: 시트 좌측 끝에서 셀 좌측 끝까지의 거리를 알려줄지 여부를 선택합니다.
2) 위쪽 읽기: 시트 상단에서 셀 상단까지의 거리르 알려줄지 여부를 선택합니다.
3) 넓이 읽기: 셀의 넓이를 알려줄지 여부를 선택합니다.
4) 높이 읽기: 셀의 높이를 알려줄지 여부를 선택합니다.

[] 셀 포인터 이름 읽기
셀 포인터 위치 셀에 이름이 지정되어 있는 경우 이름을 읽어줍니다.

[] 셀 포인터 메모 읽기
셀 포인터 위치 셀에 메모가 지정되어 있는 경우 메모 내용을 읽어줍니다.

[] 셀 포인터 링크 읽기
셀 포인터 위치 셀에 링크가 지정되어 있는 경우 링크 주소를 읽어줍니다. 스크린팁이 지정되어 있으면 스크린팁을 같이 읽어줍니다.

[] 셀 포인터 수식 읽기
셀 포인터 위치에 수식이 지정되어 있는 경우 수식을 읽어줍니다.

[] 셀 포인터 행 제목 읽기
셀 포인터 위치 셀에 해당하는 행 제목 셀이 존재하는 경우 제목 셀 내용을 읽어줍니다.

[] 셀 포인터 열 제목 읽기
셀 포인터 위치 셀에 해당하는 열 제목 셀이 존재하는 경우 제목 셀 내용을 읽어줍니다.

[] 행 처음 셀 읽기
셀 포인터가 있는 행에서 데이터가 들어 있는 첫번째 셀을 읽어줍니다.

[] 열 처음 셀 읽기
셀 포인터가 있는 열에서 데이터가 들어 있는 첫번째 셀을 읽어줍니다.

[] 행 읽기
화면상에 보이는 셀 영역에서 셀 포인터가 있는 행의 데이터를 차례로 읽어줍니다. 데이터가 없는 경우 "빈 행"이라 읽어줍니다.
-- 세부 옵션 --
1) 빈 셀 알림: 셀에 내용이 없을 경우 "빈셀"이라 알려줄지 여부를 선택합니다.

[] 열 읽기
화면상에 보이는 셀 영역에서 셀 포인터가 있는 열의 데이터를 차례로 읽어줍니다. 데이터가 없는 경우 "빈 열"이라고 읽어줍니다.
-- 세부 옵션 --
1) 빈 셀 알림: 셀에 내용이 없을 경우 "빈셀"이라 알려줄지 여부를 선택합니다.

[] 선택 영역 읽기
전체 선택영역의 주소를 알려줍니다.
-- 세부 옵션 --
1) 셀 내용 읽기 : 전체 선택 영역의 주소와 셀 내용을 같이 포함하여 알려줍니다.
앞에 읽기, 뒤에 읽기, 해제 등의 세 가지 선택 사항 중에서 한 가지 옵션을 선택합니다. 단, 셀 내용 출력 대상은 화면에 출력된 셀로
제한합니다.

[] 선택 영역 변화 읽기
현재 선택된 영역을 이전 선택영역과 비교하여 변화된 범위와 선택, 해제 정보를 읽어줍니다.
-- 세부 옵션 --
1) 셀 내용 읽기 : 선택 영역의 변화 정보와 셀 내용을 같이 포함하여 알려줍니다.
앞에 읽기, 뒤에 읽기, 해제 등의 선택 사항 중에서 한 가지 옵션을 선택합니다. 단, 셀 내용 출력 대상은 화면에 출력된 셀로 제한합니다.
2) 네용 읽기 범위 : 마지막 셀만 읽을 것인가 전체를 읽을 것인가를 선택합니다.

[] 가상 셀 포인터 주소 읽기
가상 셀 포인터의 위치를 'A1'방식으로 알려줍니다.
* 가상셀 포인터란 시트 내의 셀들을 살펴보는데 활용하기 위해 센스리더에서 내부적으로 사용하는 셀 포인터로 화면에 위치가 표시되지 않는 가상의 포인터입니다.

[] 가상 셀 포인터 내용 읽기
가상 셀 포인터 위치의 셀 내용을 읽어줍니다.
-- 세부 옵션 --
1) 빈 셀 알림: 셀에 내용이 없을 경우 "빈셀"이라 알려줄지 여부를 선택합니다.
2) 수식 알림: 수식이 있는 경우 "수식셀"이라 알려줄지 여부를 선택합니다.
3) 이름 알림: 이름이 등록되어 있는 경우 "이름셀"이라 알려줄지 여부를 선택합니다.
4) 메모 알림: 메모가 지정되어 있는 경우 "메모셀"이라 알려줄지 여부를 선택합니다.
5) 링크 알림: 링크가 지정되어 있는 경우 "링크셀"이라 알려줄지 여부를 선택합니다.

[] 가상 셀 포인터 표시형식 읽기
가상 셀 포인터의 표시형식(숫자, 통화, 날짜, 시간, 회계, 백분률 등)을 읽어줍니다.

[] 가상 셀 포인터 맞춤 읽기
가상 셀 포인터의 맞춤 형식을 읽어줍니다.
-- 세부 옵션 --
1) 가로 정렬 읽기: 가로 정렬 정보를 읽어줄지 여부를 선택합니다.
2) 세로 정렬 읽기: 세로 정렬 정보를 읽어줄지 여부를 선택합니다.
3) 들여쓰기 읽기: 들여쓰기 정보를 읽어줄지 여부를 선택합니다.
4) 텍스트 방향 읽기: 텍스트 회전 도수를 읽을지 여부를 선택합니다.
5) 텍스트 줄 바꿈 읽기: 텍스트 줄바꿈이 선택되어 있는 경우 이를 알려줄지 여부를 선택합니다.
6) 셀에 맞춤 읽기: 셀에 맞춤이 선택되어 있는 경우 이를 알려줄지 여부를 선택합니다.
7) 읽기 순서 읽기: 좌에서 우로, 우에서 좌로 등 텍스트 읽기 순서를 알려줄지 여부를 선택합니다.

[] 가상 셀 포인터 글꼴 읽기
가상 셀 포인터 위치 셀에 지정된 글꼴을 읽어줍니다.
-- 세부 옵션 --
1) 글꼴 이름 읽기: 글꼴의 이름(돋움, 돋움체, 바탕, 굴림, 굴림체 등)을 읽어줄지 여부를 선택합니다.
2) 글꼴 스타일 읽기: 글꼴 스타일(기울임꼴, 굵게, 굵은 기울임꼴) 등을 읽을지 여부를 선택합니다.
3) 글꼴 크기 읽기: 글꼴의 크기(포인트)를 읽을지 여부를 선택합니다.
4) 취소선 읽기: 취소선이 선택된 경우 이를 알려줄지 여부를 선택합니다.
5) 아래 첨자 읽기: 아래 첨자가 선택된 경우 이를 알려줄지 여부를 선택합니다.
6) 위 첨자 읽기: 위 첨자 읽기가 선택된 경우 이를 알려줄지 여부를 선택합니다.
7) 밑줄 읽기: 밑줄이 설정된 경우 밑줄 종류를 읽어줄지 여부를 선택합니다.
8) 글꼴 색 읽기: 글꼴 색을 읽어줄지 여부를 선택합니다.

[] 가상 셀 포인터 태두리 읽기
가상 셀 포인터 위치 셀의 태두리 정보를 읽어줍니다.
-- 세부 옵션 --
1) 태두리 색 읽기: 태두리에 지정된 색을 읽어줄지 여부를 선택합니다.

[] 가상 셀 포인터 무늬 읽기
가상 셀 포인터 위치 셀의 무늬 정보를 읽어줍니다.
-- 세부 옵션 --
1) 배경색 읽기: 배경색을 읽어줄지 여부를 선택합니다.

[] 가상 셀 포인터 위치 읽기
가상 셀 포인터 위치 셀의 위치(시트 좌상단에서 셀 좌상단까지의 거리)와 넓이, 높이 정보를 포인트 단위로 읽어줍니다.
-- 세부 옵션 --
1) 왼쪽 읽기: 시트 좌측 끝에서 셀 좌측 끝까지의 거리를 알려줄지 여부를 선택합니다.
2) 위쪽 읽기: 시트 상단에서 셀 상단까지의 거리르 알려줄지 여부를 선택합니다.
3) 넓이 읽기: 셀의 넓이를 알려줄지 여부를 선택합니다.
4) 높이 읽기: 셀의 높이를 알려줄지 여부를 선택합니다.

[] 가상 셀 포인터 이름 읽기
가상 셀 포인터 위치 셀에 이름이 지정되어 있는 경우 이름을 읽어줍니다.

[] 가상 셀 포인터 메모 읽기
가상 셀 포인터 위치 셀에 메모가 지정되어 있는 경우 메모 내용을 읽어줍니다.

[] 가상 셀 포인터 링크 읽기
가상 셀 포인터 위치 셀에 링크가 지정되어 있는 경우 링크 주소를 읽어줍니다.

[] 가상 셀 포인터 수식 읽기
가상 셀 포인터 위치에 수식이 지정되어 있는 경우 수식을 읽어줍니다.

[] 가상 셀 포인터 행 제목 읽기
가상 셀 포인터 위치 셀에 해당하는 행 제목 셀이 존재하는 경우 제목 셀 내용을 읽어줍니다.

[] 가상 셀 포인터 열 제목 읽기
가상 셀 포인터 위치 셀에 해당하는 열 제목 셀이 존재하는 경우 제목 셀 내용을 읽어줍니다.

[] 가상 셀 포인터 왼쪽으로 이동
가상 셀 포인터를 왼쪽으로 한 셀 이동합니다. 셀이 병합된 경우 병합된 셀 전체를 한 셀로 취급합니다.

[] 가상 셀 포인터 위로 이동
가상 셀 포인터를 위쪽으로 한 셀 이동합니다. 셀이 병합된 경우 병합된 셀 전체를 한 셀로 취급합니다.

[] 가상 셀 포인터 오른쪽으로 이동
가상 셀 포인터를 오른쪽으로 한 셀 이동합니다. 셀이 병합된 경우 병합된 셀 전체를 한 셀로 취급합니다.

[] 가상 셀 포인터 아래로 이동
가상 셀 포인터를 아래쪽으로 한 셀 이동합니다. 셀이 병합된 경우 병합된 셀 전체를 한 셀로 취급합니다.

[] 셀 포인터를 가상 셀 포인터로 이동
셀 포인터를 가상 셀 포인터 위치로 이동합니다.

[] 가상 셀 포인터를 셀 포인터로 이동
가상 셀 포인터를 셀 포인터 위치로 이동합니다.

[] 개체정보 읽기
셀 포인터 위치에 개체(그림, 버튼, 체크상자, 콤보상자) 등이 있는 경우 이를 알려줍니다.

[] 개체 목록
시트 내의 모든 개체를 목록으로 보여줍니다.
-- 팝업메뉴 --
클릭('Enter'): 개체 위치를 클릭합니다.
선택('Shift Enter'): 개체를 선택합니다.
이동('Ctrl Enter'): 개체가 표시되는 셀 위치로 셀 포인터를 이동합니다.

[] 모니터 셀 등록
셀 포인터 위치의 셀이나 선택 영역을 모니터셀 영역으로 추가합니다. 모니터 셀의 데이터는 '모니터 셀 읽기'나 '모니터 셀 목록'기능을 통해 언제든 확인할 수 있습니다.

[] 모니터 셀 목록
현재 문서 내의 모든 모니터 셀을 목록으로 보여줍니다. 첫번째 목록에 모니터 셀 이름과 셀 영역을, 두번째 목록에 첫번째 목록에서 선택한 모니터 셀 영역의 데이터를 보여줍니다.
-- 팝업메뉴 --
이동('Enter'): 선택한 셀 위치로 셀 포인터를 이동합니다.
이름 바꾸기('F2'): 모니터셀의 이름을 변경합니다.
삭제(' Delete'): 선택한 모니터셀을 삭제합니다.
항목 위로('Alt 위화살표'): 선택한 모니터셀을 위로 이동하여 순서를 변경합니다.
항목 아래로('Alt 아래화살표'): 선택한 모니터셀을 아래로 이동하여 순서를 변경합니다.

[] 모니터 셀 읽기(1~10)
해당 모니터 셀 데이터를 읽어줍니다. 11번째 모니터셀부터는 '모니터 셀 목록'을 통해서 확인할 수 있습니다.

[] 제목 기준셀 등록
셀 포인터 위치의 셀을 제목 기준셀로 등록합니다.

[] 제목 기준셀 목록
시트 내의 제목 기준셀을 목록으로 보여줍니다.
-- 팝업메뉴 --
이동('Enter'): 선택한 기준 셀 위치로 셀 포인터를 이동합니다.
삭제('Delete'): 기준셀을 삭제합니다.
등록: 셀 포인터 위치 셀을 기준셀로 등록합니다.

[] 링크 열기
셀 포인터 위치 셀의 링크를 열어줍니다.

[] 링크 목록
현재 시트 내의 링크를 목록으로 보여줍니다.
-- 팝업메뉴 --
열기('Enter'): 선택한 링크를 엽니다.
이동('Ctrl Enter'): 링크가 지정된 셀로 셀 포인터를 이동합니다.

[] 메모 목록
시트 내의 메모를 목록으로 보여줍니다. 원하는 목록을 선택 후 'Enter' 키를 누르면 메모가 지정된 셀 위치로 셀 포인터가 이동합니다.

[] 셀 기억
셀 포인터 셀을 임시로 기억합니다.

[] 기억된 셀로 이동
'셀 기억' 기능으로 기억한 셀 위치로 셀 포인터를 이동합니다.

[] 행 데이터 셀 목록
셀 포인터가 있는 행에서 데이터가 들어 있는 셀을 목록으로 보여줍니다. 원하는 항목을 선택 후 'Enter'키를 누르면 해당 셀 위치로 셀
포인터를 이동합니다.

[] 열 데이터 셀 목록
셀 포인터가 있는 열에서 데이터가 들어 있는 셀을 목록으로 보여줍니다. 원하는 항목을 선택 후 'Enter'키를 누르면 해당 셀 위치로 셀
포인터를 이동합니다.

[] 윈도우 데이터 셀 목록
창에 나타나 있는 셀 중에서 데이터가 들어 있는 셀을 목록으로 보여줍니다. 원하는 항목을 선택 후 'Enter'키를 누르면 해당 셀 위치로 셀 포인터를 이동합니다.

[] 영역 데이터 셀 목록
현재 영역의 셀 중에서 데이터가 들어 있는 셀을 목록으로 보여줍니다. 원하는 항목을 선택 후 'Enter'키를 누르면 해당 셀 위치로 셀
포인터를 이동합니다.

[] 참조하는 셀 목록
현재 셀을 직접 참조하는 모든 셀을 목록으로 보여줍니다. 원하는 항목을 선택 후 'Enter'키를 누르면 해당 셀 위치로 셀 포인터를
이동합니다.
만약, A1 셀에 '70', B1 셀에 '80' 그리고 C1 셀에 '=A1+B1' 이라는 수식이 입력되어 있다면 A1 셀에 위치한 다음 참조하는
셀을 확인하면 A1 셀을 참조하는 C1 셀이 목록으로 표시되고, 반대로 C1 셀에 위치한 다음 참조되는 셀을 확인하면 A1 셀과 B1 셀이
목록으로 표시됩니다.

[] 참조되는 셀 목록
현재 셀의 수식에 의해 직접 참조되는 셀을 목록으로 보여줍니다. 원하는 항목을 선택 후 'Enter'키를 누르면 해당 셀 위치로 셀 포인터를
이동합니다.

[] 시트 목록
전체 시트를 목록으로 보여줍니다.
-- 팝업메뉴 --
이동('Enter'): 선택한 시트를 활성화시킵니다.
이름 바꾸기('F2'): 시트 이름을 변경합니다.
삽입('Ctrl Insert'): 선택한 시트 앞에 새로운 시트를 삽입합니다.
추가('Ctrl+Shift Insert'): 마지막 시트 다음에 새로운 시트를 추가합니다.
삭제('Delete'): 선택한 시트를 삭제합니다.
위로('Alt 위화살표'): 시트를 한 단계 앞으로 이동합니다.
아래로('Alt 아래화살표'): 시트를 한 단계 뒤로 이동합니다.
처음으로('Alt Home'): 선택한 시트를 맨 처음으로 이동합니다.
끝으로('Alt End'): 선택한 시트를 맨 마지막으로 이동합니다.


ⅩⅦ-3-4. 엑셀, 한셀 기능키 리스트


센스리더는 EXCEL, 한셀 활용을 위해 다음의 단축키들을 제공합니다.

기능 설명 중 | 기호는 해당 단축키를 반복해서 누를 때 실행되는 기능을 의미합니다.

모니터 셀 읽기 1

모니터 셀 읽기 2

모니터 셀 읽기 3

모니터 셀 읽기 4

8. Ctrl-Shift-5
모니터 셀 읽기 5

9. Ctrl-Shift-6
모니터 셀 읽기 6

10. Ctrl-Shift-7
모니터 셀 읽기 7

11. Ctrl-Shift-8
모니터 셀 읽기 8

12. Ctrl-Shift-9
모니터 셀 읽기 9

13. Ctrl-Shift-B
선택 영역 읽기

14. Ctrl-Shift-C
셀 포인터 열 제목 읽기 | 열 처음 셀 읽기

셀 포인터 표시형식 읽기 | 셀 포인터 맞춤 읽기 | 셀 포인터 테두리 읽기

26. Shift-Alt-`(그레이브)
제목 기준 셀 목록
