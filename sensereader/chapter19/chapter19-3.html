ⅩⅨ-3-1. 사전 메뉴


센스 사전의 사전 메뉴 구성 및 세부 기능은 다음과 같습니다.

1. 영한사전<Alt+1>

사전 종류를 영한사전으로 변경하거나 검색어가 입력되어 있는 경우, 검색 사전을 '영한사전'으로 변경한 다음 검색합니다. 검색 이후 포커스는 내용표시창으로 이동됩니다.
영한사전이 선택된 경우 검색 메뉴에 숙어 검색 메뉴가 활성화됩니다.

2. 한영사전<Alt+2>

검색 사전을 한영사전으로 변경하거나 검색어가 입력되어 있는 경우 검색 사전을 '한영사전'으로 변경한 다음 검색합니다.
한영사전과 국어사전의 경우 검색 메뉴의 단어검색과 예문검색만 활성화됩니다.

3. 국어사전<Alt+3>

검색사전을 국어사전으로 변경하거나 검색어가 입력되어 있는 경우 검색 사전을 '국어사전'으로 변경한 다음 검색합니다.

4. 노래방<Alt+4>

검색사전 종류를 노래방으로 변경하거나, 검색어가 입력되어 있는 경우 검색사전을 노래방으로 변경하여 검색합니다.
노래방이 선택된 경우 검색 메뉴의 제목 검색, 가수 검색, 가사 검색 메뉴가 활성화됩니다.

5. 온라인 영영사전<Alt+5>

검색 사전을 온라인 영영사전으로 변경하거나 검색어가 입력되어 있는 경우 검색 사전을 '온라인 영영사전'으로 변경한 다음 검색합니다.
온라인 영영사전의 경우 검색 메뉴의 단어검색만 활성화되며, 인터넷이 연결되어 있지 않은 경우 검색에 제한 됩니다.

6. 위키백과<Alt+6>

검색 사전을 위키백과로 변경하거나 검색어가 입력되어 있는 경우 검색 사전을 '위키백과'로 변경한 다음 검색합니다.
위키백과의 경우 검색 메뉴의 단어검색만 활성화되며, 인터넷이 연결되어 있지 않은 경우 검색에 제한 됩니다.

7. 위키백과(영어)<Alt+7>

검색 사전을 위키백과(영어)로 변경하거나 검색어가 입력되어 있는 경우 검색 사전을 '위키백과(영어)'로 변경한 다음 검색합니다.
위키백과(영어)의 경우 검색 메뉴의 단어검색만 활성화되며, 인터넷이 연결되어 있지 않은 경우 검색에 제한 됩니다.

8. 미국식 발음 듣기<F7>

영한사전이 선택된 경우 검색된 단어를 미국식 발음으로 읽어줍니다.

9. 영국식 발음 듣기<F8>

영한사전이 선택된 경우 검색된 단어를 영국식 발음으로 읽어줍니다.

10. 종료<Alt+F4>

센스 사전 프로그램을 종료합니다.


ⅩⅨ-3-2. 검색 메뉴


센스 사전의 검색 메뉴 구성 및 세부 기능은 다음과 같습니다.

1. 단어<Ctrl+1>

검색 유형을 '단어검색'으로 변경합니다. 검색 이후 포커스는 내용표시창으로 이동됩니다.
단어 검색과 예문 검색은 영한, 한영, 국어사전 등 사전 기능에서만 활성화됩니다.

2. 숙어<Ctrl+2>

검색 유형을 '숙어검색'으로 변경합니다. 검색 이후 포커스는 내용표시창으로 이동되며, 숙어검색은 '영한사전'에서만 지원됩니다.

3. 예문<Ctrl+3>

검색 유형을 '예문검색'으로 변경합니다.

4. 제목 검색<Ctrl+1>

사전 종류가 노래방인 경우에만 활성화되며, 입력된 검색어를 노래 제목으로 검색합니다.

5. 가수 검색<Ctrl+2>

사전 종류가 노래방인 경우에만 활성화되며, 입력된 검색어를 가수 이름으로 검색합니다.

6. 가사 검색<Ctrl+3>

사전 종류가 노래방인 경우에만 활성화되며, 입력된 검색어를 노래의 가사에서 검색합니다.

7. 이전 결과<Ctrl+위 화살표>

검색 결과가 여러 개인 경우, 검색 결과의 이전 항목 내용을 표시합니다.

8. 다음 결과<Ctrl+아래 화살표>

검색 결과의 다음 항목 내용을 표시합니다. 검색 결과가 여러 개인 경우 사용합니다.

9. 뒤로<Alt+왼쪽 화살표>

이전 출력 내용을 표시합니다.

10. 앞으로<Alt+오른쪽 화살표>

다음 출력 내용을 표시합니다.

11. 자동검색<Enter>

내용출력창의 캐럿 위치 단어를 자동으로 입력 후 검색합니다.

12. 검색<Ctrl+F>

내용 출력창의 검색 내용 중 특정 문자열을 검색합니다.

13. 다시 찾기<F3>

이전의 검색 문자열을 기준으로 재검색을 실행합니다.

14. 반대로 찾기<Shift+F3>

반대 검색 방향으로 문자열을 재검색합니다.

참고사항

1. 단어/숙어/예문 검색 기능은 사전을 선택했을 때만 활성화되고, 제목/가수/가사 검색 기능은 노래방을 선택했을 때만 활성화 됩니다.

2. 센스 사전은 프로그램이 실행되는 동안 사전종류, 검색유형, 검색어, 현재 결과 항목, 캐럿 위치 등의 모든 출력 결과를 내부적으로 저장하고 있습니다.


ⅩⅨ-3-3. 보기 메뉴


센스 사전의 보기 메뉴 구성은 다음과 같습니다.

상태 표시줄

상태 표시줄을 보일지 여부를 선택합니다. 상태 표시줄에는 사전 종류와 검색 결과 개수가 표시됩니다.


ⅩⅨ-3-4. 설정 메뉴


센스 사전옵션 메뉴 구성은 다음과 같습니다.

1. 기본 영어 사전

'자동 사전 선택' 기능이 동작 중일 때 영어가 입력되면 사용할 사전을 선택합니다. '한영사전', '국어사전', '위키백과'등 자주 사용하는 사전 중에서 선택할 수 있습니다.

2. 기본 한글 사전

<
p>'자동 사전 선택' 기능이 동작 중일 때 한글이 입력되면 사용할 사전을 선택합니다. '한영사전', '국어사전' , '위키백과'등 자주 사용하는 사전 중에서 선택할 수 있습니다.

3. 자동 사전 선택

검색어로 입력된 문자 종류에 따라 검색할 사전 종류를 자동으로 변경해 주는 '자동 사전 선택' 기능의 동작을 설정합니다.
자동 사전 선택 기능은 기본적으로 영문이 입력되면 '영한사전'으로, 한글이 입력되면 '기본 한글 사전'에서 선택한 사전으로 전환됩니다.

다음 세 가지 옵션 중 하나를 선택합니다.
- 사용안함: '자동 사전 선택' 기능을 사용하지 않습니다.
- 자동검색: 센스사전의 검색어 입력창한 통한 수동 검색은 '자동 사전 선택' 기능을 사용하지 않고, 자동검색의 경우에만 '자동 사전 선택' 기능이 동작하도록 설정합니다.
- 모두: 자동검색, 수동검색 모두 '자동 사전 선택' 기능이 동작하도록 설정합니다.

4. 자동 발음 듣기

'영한 사전'이 선택된 경우 검색어 발음 듣기 기능의 설정을 조정합니다.

다음 세 가지 옵션 중 하나를 선택합니다.
- 사용안함 : 발음듣기 기능을 사용하지 않습니다.
- 미국식 : 미국식으로 영어 단어를 발음합니다.
- 영국식 : 영국식으로 영어단어를 발음합니다.

{{5. 결과 목록에 미리보기 사용}}

<검색 결과> 목록에 검색된 내용의 일부분을 미리 표시할지 여부를 선택 합니다.


{{6. 이스케이프로 최소화}}

<ESC> 키를 사용해 창을 최소화할지 여부를 선택합니다.


{{7. 백그라운드로 실행}}

프로그램 창을 숨길지 여부를 선택합니다. 창이 숨겨진 경우 트레이 아이콘을 사용하거나 센스리더의 '사전 검색' 기능을 사용해 창을 활성화시킬 수 있습니다. 창이 최소화 되거나 다른 프로그램이 활성화되면 다시 창이 숨겨집니다.
