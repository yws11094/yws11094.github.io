ⅩⅧ-1. 보조 음성 엔진의 활용


프로페셔널과 더 넥스트 사용자의 경우 센스리더를 설치하신 후 다음과 같은 작업을 통해 음성 엔진을 추가설치 하실 수 있습니다.


1. 보조 음성 엔진 추가 설치

  1. <Ctrl-\(백슬래시)> 키를 눌러 센스리더 윈도우를 활성화 합니다.
  2. 아래 방향키로 [2. 센스리더 도구] 항목을 찾아 <Enter> 키를 누른 후, 아래 방향키로 [음성 엔진 설치 관리자]를 찾아 실행합니다.
  3. '수정'라디오 버튼을 선택하신 후 '다음'버튼을 실행합니다.
  4. '트리뷰'음성이 출력되면 <위, 아래 방향키>를 이용하여 원하는 음성엔진을 찾으신 후 <Space Bar>키로 선택합니다.
  5. '다음'버튼을 눌러 설치를 진행 합니다.
  6. '완료'버튼을 눌러 설치를 종료 합니다.

2. 보조 음성 엔진 선택

보조 음성 엔진을 설정하기 위해 <Ctrl-\(백슬래시)>키를 눌러 센스리더 윈도우를 열고 <Alt>키를 눌러 [일반] - [보조음성 엔진 선택]을 실행합니다.
'보조 음성 엔진 선택'대화상자가 호출되면 언어별 보조 음성 엔진의 선택과 자동모드 사용 엔진의 설정 그리고 보조 음성 엔진의 속도, 높이, 크기 등을 지정합니다.

'보조 음성 엔진 선택' 대화상자의 구성은 다음과 같습니다.
1) 언어 선택 목록상자
현재 컴퓨터에서 사용할 수 있는 언어 종류가 목록으로 표시됩니다. 특정 언어를 자동 모드에서 사용하려면 해당 언어에 위치한 이후 <Space> 키를 눌러 자동 모드로 선택해 주어야 합니다.
2) 음성엔진 선택 목록상자
언어 선택 목록상자에서 언어를 선택한 이후 음성 엔진 목록으로 이동하면 현재 컴퓨터에서 사용할 수 있는 음성엔진이 목록으로 표시됩니다. 사용을 원하는 음성 엔진을 방향키를 사용하여 선택합니다.
3) 속도, 높이, 크기 설정 스크롤바
음성엔진을 선택한 다음 <Tab> 키를 누르면 속도, 높이, 크기 등을 설정할 수 있는 스크롤바로 이동합니다. 방향키를 사용하여 사용자가 원하는 값으로 설정합니다.
4) 확인 버튼
'확인' 버튼을 눌러 사용자가 지정한 설정을 적용합니다.

보조 음성 엔진 선택 과정은 센스리더 설치 이후 한 번만 진행하면 됩니다.


3. 보조 음성 엔진 모드 변경 <Insert-Ctrl-E>

보조 음성 엔진의 동작을 캐럿 자동모드/전체 자동 모드/수동 모드/해제 중에서 선택합니다.

- 자동 모드: 출력 언어를 자동 판단하여 해당 엔진으로 출력합니다. 자동 모드에서 사용할 엔진은 <보조음성엔진 선택> 대화상자를 이용하여
자동모드 사용 엔진으로 선택해 두어야 합니다. 예를 들어, <보조음성엔진 선택> 대화상자에서 영어엔진을 자동모드 사용 엔진으로 설정하였다면
한/영 혼용 문장에서 한글은 기본 음성엔진이 동작하고 영어는 영어 보조 음성엔진이 동작하게 됩니다. 해당 엔진이 없으면 기본엔진으로 출력됩니다.

* 캐럿 자동 모드에서는 가상커서의 내용과 편집기에서의 캐럿 이동에 대한 내용에 대해서 보조 음성 엔진을 적용하고, 전체 자동 모드는 메뉴 및 일반 메시지 내용 모두에 보조 음성 엔진을 적용합니다.

- 수동 모드: 출력엔진을 자동 변경하지 않고 <보조음성엔진 선택> 대화상자에서 지정된 엔진으로 출력합니다. 단, 인터넷에서 lang 속성이
지정되어 있는 경우는 lan 속성으로 지정된 언어의 엔진으로 출력됩니다.

- 해제: 보조음성엔진을 사용하지 않습니다.
