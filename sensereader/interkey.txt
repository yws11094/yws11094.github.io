0. 브라우저 탭키 환경이란

센스리더 4.0의 인터넷 가상커서(이하 가상커서)에서는 '브라우저 탭키'라는 환경이 추가되었습니다.
'브라우저 탭키' 환경이란 인터넷 요소 이동을 위한 탭(tab) 키와 웹페이지 탭을 이동하기 위한 콘트롤 + 탭(ctrl + tab) 키 밑 콘트롤 숫자(ctrl + 1 2 3 …) 키를 가상커서에서 처리하지 않고 브라우저가 직접 처리하게 함으로서 인터넷 탐색 시 브라우저 고유의 기능을 차단하지 않고 가상커서와 함께 사용할 수 있도록 하는 환경입니다.
또한 '브라우저 탭키' 환경에서는 가상커서에서 웹 탐색을 도와주는 기능들에 대해 영문 글자와 같이 간편한 기능키들을 새롭게 정의하였습니다.
기존의 가상커서 환경에 익숙한 사용자를 위해 '가상커서 탭키' 환경을 함께 제공하고 있습니다.
'가상커서 탭키' 환경에 대한 자세한 설명은 센스리더 도움말의 "※ Ⅹ장 인터넷의 활용(가상커서)"편을 참고해 주십시오.

1. 브라우저 탭키 환경에서의 변경된 기능키 목록

1.1 개념 : '브라우저 탭키' 환경에서 변경된 기능키에 대한 개념은 아래 몇가지로 요약할 수 있습니다.
(1) 앞으로 탐색은 키 하나	
(2) 뒤로 탐색은 쉬프트(shift) + 키 하나
(3) 링크, 목록항목, 헤딩과 같이 한 줄에 표현되는 정보들은 오른손 키 배열 사용.(ex : i h j k l n m)
(4) 테이블, 프레임, 외부객체와 같이 블록 형태의 정보는 왼손 키 배열 사용.(ex : w e r t s d f b)
(5) 프레임 목록(g), 링크 목록(;), 마크 목록(,) 키는 프레임 이동(f), 링크 이동(l), 마크 이동(m) 오른쪽에 위치.
(6) 편집창 이동, 다음 문자열 찾기, 텍스트 이동은 같은 키그룹인 F2 F3 F4 에 지정	
(7) 기존 탭(Tab), 쉬프트+탭(shift+Tab) 및 마크이동(ctrl+1 2 3 …) 키는 인서트+탭(insert+tab), 인서트+쉬프트+탭(insert+shift+tab) 및 인서트+숫자(insert+1 2 3 …) 키로 대체

2. 가상커서에서의 입력모드와 기능키 모드	

2.1 기능키 모드 전환 : 가상커서에서 탭(tab) 키로 편집창을 만나게 되면(멀티라인 편집창 포함) 기본 설정에 의해 글자를 누르면 바로 입력이 되는 입력모드로 변경이 되어 링크 이동(l), 헤딩 이동(h)과 같은 기능키를 사용할 수 없게 됩니다.
'입력 모드' 상황에서 기능키를 사용하고자 할 때에는 탭(tab) 키로 다른 위치에서 기능을 사용할 수도 있지만,
이스케이프(esc) 키를 눌러 '기능키 모드'로 전환한 후에 원하는 기능키를 사용할 수도 있습니다.
편집창에서 이스케이프(esc) 키를 누르게 되면 "입력 종료"라는 메시지와 함께 편집창은 "해제 편집창"으로 음성출력하여 지금이 기능키 모드임을 알려줍니다.
'기능키 모드'에서 글자를 입력하면 비프음	(Beep)이 발생하며 입력은 무시됩니다.

2.2 입력 모드 전환 : '기능키 모드'에서 다시 입력모드로 전환하려면 아래의 몇가지 방법을 사용하면 됩니다.
(1) 가상커서 설정에서 "가상커서시 편집창 입력" 항목(※ 6. 브라우저 탭키 환경 기본 설정 값)이 선택되어 있는 기본 상황에서는 탭(tab) 키, 쉬프트+탭(shift+tab) 키 사용	
(2) 스페이스(space) 키 사용
(3) 편집작업을 의미하는 백스페이스(backspace), 좌우 화살표(← →), 홈(home), 엔드(end), 딜리트(delete) 및 콘트롤+딜리트(ctrl+delete) 키를 누르면 해당 기능이 동작하며 입력모드로 전환
가상커서가 '기능키 모드'에서 '입력 모드'로 전환이 되면 "입력 시작"이라는 메시지와 함께 "해제 편집창"은 "편집창"으로 알려주게 됩니다.

2.3 모드 자동 변환 : 이전 편집창 이동(shift+F2), 다음 편집창 이동(F2) 기능을 사용하면 가상커서 설정에 관계없이 편집창은 "입력 모드"가 됩니다.
이전 컨트롤 이동(shift+n), 다음 컨트롤 이동(n) 을 사용하면 가상커서 설정과 관계 없이 편집창은 "기능키 모드"로 전환됩니다.
"가상커서시 편집창 입력" 항목(※ 6. 브라우저 탭키 환경 기본 설정 값)이 선택된 경우 탭(tab) 키와 쉬프트+탭(shift+tab) 키를 누르면 자동으로 "입력 모드"로 전환됩니다.
"가상커서시 편집창 입력" 항목(※ 6. 브라우저 탭키 환경 기본 설정 값)이 해제된 경우 탭(tab) 키와 쉬프트+탭(shift+tab) 키를 누르면 자동으로 "기능키 모드"로 전환됩니다.

3. 멀티라인 편집창 사용	

3.1 편집창과 동일한 멀티라인 편집창 : 멀티라인 편집창에서 첫번째 줄은 "※ 2. 가상커서에서의 입력모드와 기능키 모드" 처럼 동작하여 가상커서 상태에서도 바로 편집작업을 할 수 있게 되며,
입력된 내용이 두 줄 이상일 때에는 두번 째 줄 부터는 "해제 멀티라인 편집창" 이라고 알려주어 입력할 수 없는 멀티라인 편집창임을 알려줍니다.

3.2 해제 멀티라인 편집창 : "해제 멀티라인 편집창"에서 스페이스(space), 엔터(enter) 키를 누르게 되면 가상커서 사용은 임시로 해제가 되어(※ 4. 가상커서 임시 해제) 화살표(↑ ↓ ← →) 키로 이동하며 자유롭게 편집을 할 수 있게 되고,
가상커서가 임시로 해제가 되었을 때 이스케이프(escape), 탭(tab), 쉬프트+탭(shift+tab) 키를 눌러 가상커서 사용을 선택할 수 있습니다.
"해제 멀티라인 편집창"에서 글자를 입력하면 비프음(Beep)이 발생하며 입력은 무시가 됩니다.

4. 가상커서 임시 해제

4.1 해제하기 : 웹페이지를 탐색하는 중에 채크상자(Check box), 라디오버튼(Radio button), 콤보상자(Combo box) 등 엔터(enter) 키를 누르면 가상커서가 해제되는 요소에 더하여
가상커서 어디서나 콘트롤+위 화살표(ctrl+↑)와 콘트롤+아래 화살표(ctrl+↓)를 누르게 되면 "가상커서 사용 해제"라는 메시지와 함께 가상커서는 임시로 해제되게 됩니다.
가상커서가 해제된 상황에서는 모든 입력은 브라우저에 의해 처리가 되어 메뉴 항목 이동, 라디오버튼 변경, 멀티라인 편집창의 현재 위치 변경 및 플레시 객체 조작 등을 할 수 있게 됩니다.
4.2 가상커서 즉시 사용 : 가상커서가 임시로 해제가 된 상태에서 이스케이프(escape), 탭(tab) 그리고 쉬프트+탭(shift+tab) 키를 누르면 "가상커서 사용 선택"이라는 메시지와 함께 가상커서가 자동으로 켜지게 됩니다.
가상커서가 임시로 해제된 상태에서는 페이지가 변경되거나 다른 주소로 이동하여도 가상커서가 자동으로 켜지지 않게 되어 사용자의 작업이 가상커서에 의해 차단되지 않게 됩니다.

5. 가상커서 설정 대화상자(ctrl+shift+F9) 기능

5.1 변경내용 적용(A)
가상커서 사용 중 숨김내용 표시 변경(ctrl+ /), 미디어 이동시간 변경('), 자동 포커스 사용(ctrl+ .) 기능(※ 1.2 변경된 키)을 기능키로 사용하게 되면 현재 환경에는 반영이 되지만 설정 내용이 저장되지는 않습니다.
따라서 가상커서의 설정값을 기능키로 변경을 하였을 때 가상커서 설정 대화상자(ctrl+shift+F9)를 실행하면 "변경 내용 적용(A)" 버튼이 처음에 활성이 되어 변경된 설정을 바로 저장하고 빠져나올 수 있게 합니다.
"변경 내용 적용(A)" 버튼은 "가상커서 설정 대화상자"를 닫으면 다른 설정이 변하기 전 까지 비활성화 상태가 됩니다.
이는 설정을 저장할 의사가 없는 경우에도 "변경 내용 적용(A)" 버튼이 자리하고 있어 다른 기능을 사용하기 위해 키를 한 번 더 눌러야 하는 불편을 없애기 위함 입니다.

5.2 설정(S)
"링크 속성 읽기", "자동 포커스 사용", "가상커서시 편집창 입력" 등 가상커서 사용을 위한 28가지의 설정(※ 6. 브라우저 탭키 환경 기본 설정 값)을 할 수 있습니다.
각 항목에서 스페이스(space) 키 또는 마우스 왼쪽 클릭으로 값을 변경할 수 있고
"확인" 버튼을 눌러 설정을 저장하거나
"취소" 버튼을 눌러 설정을 취소하고 가상커서로 돌아올 수 있습니다.

5.3 키 설정(K)
가상커서에서 사용되는 171가지의 기능(※ 1.2 변경된 키)에 대한 기능키를 지정하거나 확인할 수 있는 대화상자가 실행됩니다.
목록에서 원하는 기능을 선택하고 "등록(R)" 버튼을 눌러 기능키를 지정할 수 있고
"삭제(D)" 버튼을 눌러 기능키를 삭제할 수 있습니다.
"확인" 버튼을 눌러 기능키 설정을 저장하거나
"취소" 버튼을 눌러 설정을 취소하고 가상커서로 돌아올 수 있습니다.

5.4 가상커서 탭키 사용(V)
기존의 기능키와 설정값으로 가상커서를 사용할 수 있는 환경(※ 센스리더 도움말 Ⅹ장 인터넷의 활용(가상커서))으로 전환하는 기능입니다.
"예(Y)" 버튼을 눌러 환경을 전환하거나
"아니오(N)" 버튼을 눌러 가상커서로 돌아올 수 있습니다.
'가상커서 탭키' 환경에서는 브라우저의 탭(tab) 키를 사용하려면 인서트+탭(insert+tab) 키를 눌러 사용할 수 있습니다.

5.5 브라우저 탭키 사용(B)
브라우저의 고유 기능(탭 이동, 창 이동)들과 가상커서 기능을 같이 사용할 수 있는 '브라우저 탭키' 환경(※ 0. 브라우저 탭키 환경이란)으로 전환합니다.
"예(Y)" 버튼을 눌러 환경을 전환하거나
"아니오(N)" 버튼을 눌러 가상커서로 돌아올 수 있습니다.
'브라우저 탭키' 환경에서는 가상커서의 탭(tab) 키를 사용하려면 인서트+탭(insert+tab) 키를 눌러 사용할 수 있습니다.

5.6 현재 환경 초기화(I)
현재의 환경 '가상커서 탭키' 또는 '브라우저 탭키' 에 대하여 설정값과 기능키를 초기값으로 설정합니다.
"예(Y)" 버튼을 눌러 환경을 초기화 하거나
"아니오(N)" 버튼을 눌러 가상커서로 돌아올 수 있습니다.
'초기화' 작업은 현재 지정된 환경에만 적용이 되며 나머지 환경 또는 센스리더의 프로그램별 환경에는 영향을 주지 않습니다.

5.7 닫기(C)
"가상커서 설정 대화상자"를 닫고 가상커서로 돌아갑니다.
이 때, "변경 내용 적용(A)" 버튼이 비활성화 됩니다.
